## 目录结构

```
工作经验/
│
├── 周记整理/          ← 原始笔记 + 加工分析（按周时序）
│   ├── 0424工作第三周工作整理.txt
│   ├── 0501第四周工作整理.txt
│   └── 第四周工作整理-分析与优化.md
│
├── 方法论与SOP/        ← 可长期复用的流程/原则文档
│   ├── Plan-Act AI协作SOP.md
│   └── AI结合工程.txt
│
├── 速查手册/           ← 高频查阅的模板、检查清单
│   └── Prompt模板速查手册.md
│
└── 工程实践/           ← 技术问题分析、架构设计记录
    （待积累）
```

## 文件分类规则

| 目录 | 放什么 | 不放什么 |
|------|--------|---------|
| **周记整理** | 以周为单位的原始笔记 + 基于笔记的加工分析文档 | 不按周整理的、跨周的技术总结 |
| **方法论与SOP** | 可长期复用、不受时间影响的工作流程/原则 | 已经过时的、仅针对某周的临时方案 |
| **速查手册** | 写代码时高频查阅的模板/清单/参考卡 | 长篇分析文章、完整方法论文档 |
| **工程实践** | 具体技术问题的分析、架构设计决策、踩坑记录 | 工作周记、通用方法论 |
