# 功能提示小助手 — 业务流程文档

## 概述

功能提示小助手是主界面（MainUI）上的一个双行提示系统，用于向玩家展示当前可操作的关键信息，并提供快捷跳转入口。

**代码位置**: `Assets/Scripts/Hot/Logic/UI/Main/MainUIController_Assistant.cs`（MainUIController 分部类）

---

## UI 布局

小助手在主界面上占据两个独立的按钮行：

| 行 | 按钮 | 文本 | 图标 | 用途 |
|----|------|------|------|------|
| 底部行 | `btnAssistant` | `txAssistant` | 无 | 类型 1 — 可建造提示 |
| 顶部行 | `btnAssistant2` | `txAssistant2` | `imgIconAssistant2` | 类型 2-5 — 缺货提示（未来扩展 6、7） |

两行独立显示/隐藏，可同时出现。

---

## 类型体系

### 类型 1 — 建造提示（底部行 `btnAssistant`）

- **触发条件**: 存在可建造的建筑（`BuildingSystem.GetConstructableBuildingIdsForMap()` 返回数量 > 0）
- **文本**: 配置表 `assistant` 表 Type=1，回退 `text_info` ID 900520
- **点击行为**: 打开建造 UI（`BuildUI.Create`）
- **优先级**: 独立显示行，不与其他类型互斥
- **刷新时机**: `OnItemCountChanged`、`OnBuildingLevelUpSuccess`、`OnSyncMapData`

### 类型 2-5 — 生产缺货提示（顶部行 `btnAssistant2`）

按优先级从高到低互斥显示，同时只显示一条：

| 优先级 | 类型 | 建筑 | 建筑枚举 | 页面跳转 ID | 文本回退 ID |
|--------|------|------|----------|-------------|-------------|
| 1（最高）| 5 | 鬼火锅（体力）| E_111 | 1002 | 900524 |
| 2 | 4 | 纸扎庐（精力）| E_110 | 1003 | 900523 |
| 3 | 3 | 孟茶摊（心情）| E_109 | 1004 | 900522 |
| 4（最低）| 2 | 参王汤（血量）| E_102 | 1001 | 900521 |

- **触发条件**:
  1. 建筑等级 > 0
  2. 该建筑最新解锁的产品库存为 0
- **文本**: 配置表 `assistant` 表对应 Type，回退 `text_info` 对应 ID
- **文本占位符**: `##0##` 替换为缺货物品名称
- **图标**: 显示缺货物品的图标（`itemCfg.Icon`）
- **点击行为**: 跳转到对应建筑的页面（`GotoView.Go(pageJumpId)`）
- **刷新时机**: 同类型 1

## 文本获取流程

```
GetAssistantHintText(assistantType, placeholderItemId)
  │
  ├─ 1. 查 assistant 配置表（Type 匹配）
  │     └─ 找到且有文本 → 进入格式化
  │
  ├─ 2. 回退 text_info 表
  │     ├─ 类型1 → 900520
  │     ├─ 类型2 → 900521
  │     ├─ 类型3 → 900522
  │     ├─ 类型4 → 900523
  │     └─ 类型5 → 900524
  │
  └─ 3. 格式化：##0## → 物品名称
```

---

## 刷新机制

`RefreshAssistantHints()` 在以下事件触发时调用：
- `OnItemCountChanged` — 物品数量变化
- `OnBuildingLevelUpSuccess` — 建筑升级成功
- `OnSyncMapData` — 地图数据同步

每次刷新：
1. 先判断类型 1（建造）是否显示，设置底部行
2. 再按优先级判断类型 2-5（缺货）显示哪一条，设置顶部行
3. 分别设置两行的 `GameObject.SetActive`

## 引导集成

- 建造按钮（`btnAssistant`）注册了引导触发：`GuideTouchId = "click_btnAssistant"`
- 引导系统可通过 `GuideTriggerType.Click` 在此按钮上触发引导手指
