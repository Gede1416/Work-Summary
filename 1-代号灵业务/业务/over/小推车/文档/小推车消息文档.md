# 小推车（Cart）网络消息文档

## 数据结构

### CartDataContainer（推车完整数据）

| 字段 | 类型 | 说明 |
|------|------|------|
| slot_list | repeated CartSlotData | 上架槽位列表 |
| total_sold_count | int64 | 累计已售出份数，用于可领取提示阈值(P_1090)判断 |

### CartSlotData（单个槽位）

| 字段 | 类型 | 说明 |
|------|------|------|
| slot_index | uint32 | 槽位序号，增量同步定位用 |
| product_item_id | uint32 | 上架商品物品id（item类型6），0表示空闲 |
| count | int32 | 当前库存数量 |
| income | int64 | 该槽位累计待领取收益（银元） |

---

## 服务端推送（Client 被动接收）

### SCInitCartInfo — 初始化推车数据

服务端在进入游戏时主动推送完整推车数据。

| 字段 | 类型 | 说明 |
|------|------|------|
| user_cart_data | CartDataContainer | 完整推车数据 |

客户端自动处理：`cartContainer.ResetFromProto()` 全量覆盖本地数据。

### SCSyncCartSlotData — 增量同步槽位

上架/下架/一键上架/一键补货等服务端计算结果后，服务端推送变化的槽位。

| 字段 | 类型 | 说明 |
|------|------|------|
| slot_list | repeated CartSlotData | 变化的槽位数据列表 |

客户端自动处理：`cartContainer.ApplySyncSlot()` 按 slot_index 定位增量更新。

---

## 客户端请求（Client → Server）

所有请求方法位于 `CarSystem.Req*` 系列。

### NPC 购买

```csharp
// NPC 到达小推车购买一个商品
CarSystem.Ins.ReqCartBuy(
    slotIndex: 0,                          // 购买的槽位序号
    onSuccess: (rsp) => {
        Debug.Log($"售价: {rsp.Price}");    // 本次售出单价(银元)，客户端飘字用
    },
    onFail: (head) => {
        Debug.LogError($"购买失败: {head.ErrorCode}");
        return true;                       // true = 已处理错误
    }
);
```

> 服务端计算售价，扣减该槽位库存1，累计收益与售出份数。购买后会推送 `SCSyncCartSlotData`。

### 上架商品

```csharp
// 从百炼炉库存(玩家物品)中扣除，转入指定槽位
CarSystem.Ins.ReqCartListProduct(
    slotIndex: 0,                          // 目标槽位序号
    productItemId: 10001,                  // 上架商品物品id（item类型6）
    count: 10,                             // 上架数量（实际容量=MIN(堆叠上限, 建筑库存上限)）
    onSuccess: (rsp) => {
        Debug.Log("上架成功");
    }
);
```

> 上架成功后推送 `SCSyncItemData`（同步玩家物品）+ `SCSyncCartSlotData`（同步槽位）。

### 下架商品

```csharp
// 下架指定槽位，剩余库存返还为玩家物品
CarSystem.Ins.ReqCartUnListProduct(
    slotIndex: 0,                          // 待下架的槽位序号
    onSuccess: (rsp) => {
        Debug.Log("下架成功");
    }
);
```

> 下架成功后推送 `SCSyncItemData` + `SCSyncCartSlotData`。

### 一键最优上架

```csharp
// 计算玩家所有商品(item类型6)的秒金，从高到低依次填满空闲槽位
CarSystem.Ins.ReqCartOneKeyList(
    onSuccess: (rsp) => {
        Debug.Log("一键上架完成");
    }
);
```

> 成功后推送 `SCSyncItemData` + `SCSyncCartSlotData`。

### 一键补货

```csharp
// 对所有已上架槽位，从玩家商品库存补充同种商品至容量上限
CarSystem.Ins.ReqCartOneKeyRefill(
    onSuccess: (rsp) => {
        Debug.Log("一键补货完成");
    }
);
```

> 成功后推送 `SCSyncItemData` + `SCSyncCartSlotData`。

### 收益领取

```csharp
// 领取累计收益，待领取收益清零并转入玩家背包
CarSystem.Ins.ReqCartCollectIncome(
    onSuccess: (rsp) => {
        Debug.Log($"领取收益: {rsp.Income} 银元");
    }
);
```

> 成功后推送 `SCSyncItemData`。

## 数据查询

```csharp
// 获取所有槽位
var slots = G.userData.cartContainer.SlotList;

// 获取累计售出份数
var soldCount = G.userData.cartContainer.TotalSoldCount;

// 查询特定槽位
var slot = slots.Find(s => s.SlotIndex == targetIndex);
if (slot != null)
{
    Debug.Log($"槽位{slot.SlotIndex}: 商品{slot.ProductItemId} x{slot.Count} 收益{slot.Income}");
}
```
