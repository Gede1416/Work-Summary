# Markdown 语法总结

## 一、Markdown 简介

**Markdown** 是一种轻量级标记语言，由 John Gruber 于 2004 年创建。它使用纯文本格式编写文档，语法简洁直观，可轻松转换为 HTML、PDF 等多种格式。Markdown 广泛应用于：

- 编写 README 文档（GitHub、GitLab 等）
- 撰写博客文章
- 做笔记（如 Obsidian、Notion、VS Code 内置 Markdown 预览）
- 文档协作与版本控制

核心设计理念：**易读易写**，即使在纯文本状态下也能保持良好的可读性。

## 二、基础语法

### 1. 标题

使用 `#` 号表示标题，`#` 的数量代表标题级别（1~6 级）。

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

### 2. 段落与换行

- 段落之间用**空行**分隔。
- 行尾加**两个空格**再回车可实现强制换行。

```markdown
这是第一段。

这是第二段，与上一段之间有空行。  
这是本行的下一行（行尾有两个空格）。
```

### 3. 强调

```markdown
*斜体* 或 _斜体_
**粗体** 或 __粗体__
***粗斜体*** 或 ___粗斜体___
~~删除线~~
```

### 4. 列表

#### 无序列表

使用 `-`、`*` 或 `+`：

```markdown
- 苹果
- 香蕉
- 橘子
```

#### 有序列表

```markdown
1. 第一步
2. 第二步
3. 第三步
```

#### 嵌套列表

缩进两个空格或一个 Tab：

```markdown
- 水果
  - 热带水果
    - 芒果
  - 温带水果
    - 苹果
- 蔬菜
```

#### 任务列表

```markdown
- [x] 已完成任务
- [ ] 未完成任务
- [ ] 待办事项
```

### 5. 链接

```markdown
[显示文本](https://example.com)

[带标题的链接](https://example.com "鼠标悬停提示")

相对路径：点击查看[说明文档](./docs/guide.md)
```

### 6. 图片

```markdown
![替代文本](图片URL)

![带标题的图片](图片URL "图片标题")
```

### 7. 代码

#### 行内代码

```markdown
使用 `console.log()` 打印日志。
```

#### 代码块

用三个反引号包裹，可指定语言实现语法高亮：

````markdown
```javascript
function greet(name) {
  return `Hello, ${name}!`;
}
```
````

### 8. 引用

使用 `>` 符号：

```markdown
> 这是一段引用。
>
> 引用可以跨多行。
>> 引用可以嵌套。
```

### 9. 分割线

使用三个或以上的 `-`、`*` 或 `_`：

```markdown
---
***
___
```

## 三、扩展语法

### 1. 表格

使用 `|` 分隔列，`-` 分隔表头与内容，`:` 控制对齐方式：

```markdown
| 左对齐 | 居中对齐 | 右对齐 |
| :----- | :------: | -----: |
| 单元格 | 单元格   | 单元格 |
| 单元格 | 单元格   | 单元格 |
```

### 2. 脚注

```markdown
这是一段文字[^1]。

[^1]: 这是脚注的说明内容。
```

### 3. 自动链接

```markdown
<https://example.com>
<email@example.com>
```

### 4. 表情符号（Emoji）

部分 Markdown 编辑器支持：

```markdown
:smile: :heart: :+1:
```

## 四、HTML 混写

Markdown 支持内嵌 HTML 标签，适用于更复杂的排版：

```markdown
这是一段文字，<span style="color: red;">这部分是红色</span>。

<details>
  <summary>点击展开</summary>
  这里是隐藏的详细内容。
</details>
```

| ------------------------ | ------------------------------------ |
