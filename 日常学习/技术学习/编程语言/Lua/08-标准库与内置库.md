# Lua 标准库与内置库

## 详细描述

Lua 的标准库刻意保持精简——整个标准库只有大约 20 余个函数，分布在 7 个模块中。这种极简主义是 Lua 可嵌入性的关键：标准库越小，嵌入时越容易控制环境和安全沙箱。

**基础库**（Base Library）：提供 Lua 核心功能的函数。`print`（输出）、`type`（获取类型）、`tostring`/`tonumber`（类型转换）、`pairs`/`ipairs`（迭代器）、`error`/`assert`（错误处理）、`pcall`/`xpcall`（保护调用）、`select`（处理变长参数）、`rawget`/`rawset`（绕过元表访问）、`next`（遍历表的下一个键值对）。

**table 库**：操作 table 的核心工具集：
- `table.insert(t, pos, value)` / `table.remove(t, pos)`：插入/删除
- `table.sort(t, comp)`：排序
- `table.concat(t, sep)`：连接字符串数组（比 `..` 循环拼接高效得多）
- `table.move(a, f, e, t, dest)`（Lua 5.3+）：高效移动元素
- `table.pack(...)` / `table.unpack(t)`：打包/解包

**string 库**：丰富的字符串操作。模式匹配（Pattern Matching）是亮点——虽然不是完整的正则表达式，但简单高效，是 Lua 特色的字符串处理方式。模式元素包括 `.`（匹配任意字符）、`%a`（字母）、`%d`（数字）、`%s`（空白）、`%w`（字母数字）、`*`（0或多个）、`+`（1或多个）、`-`（0或多个，尽可能少匹配）、`?`（0或1个）。捕获用 `()` 表示。

**math 库**：标准数学函数（`sin`、`cos`、`sqrt`、`log`、`exp`等）和常量（`math.pi`、`math.huge`）。注意 `math.random` 和 `math.randomseed` 的组合——Lua 5.4 改进了随机数生成器。

**io 库**：简单 I/O 模型（`io.input`、`io.output`、`io.write`、`io.read`）和完整 I/O 模型（`io.open` 返回文件句柄，用 `file:read`、`file:write`、`file:close`）。完整模型更安全可靠。

**os 库**：操作系统交互：`os.time()`（当前时间戳）、`os.date()`（格式化日期）、`os.execute(cmd)`（执行系统命令）、`os.tmpname()`（临时文件名）、`os.exit()`（退出进程）、`os.rename()`（重命名文件）、`os.remove()`（删除文件）。

**debug 库**：用于内省（introspection）和调试的工具。`debug.traceback()`（调用栈）、`debug.getinfo()`（函数信息）、`debug.getlocal()`（局部变量）、`debug.setmetatable()`（设置元表）。在生产代码中应谨慎使用。

**coroutine 库**：协程创建和控制（见第 7 节）。

## 代码实例

```lua
-- ========== 1. Table 库 ==========

-- table.insert / remove
local fruits = {"banana", "cherry"}
table.insert(fruits, 1, "apple")     -- {"apple", "banana", "cherry"}
table.insert(fruits, "date")         -- 末尾追加
local last = table.remove(fruits)     -- 移除（返回 "date"）
local first = table.remove(fruits, 1) -- 移除第一个

-- table.sort：自定义排序
local scores = {
    {name = "Alice", score = 95},
    {name = "Bob", score = 87},
    {name = "Charlie", score = 92},
}
table.sort(scores, function(a, b)
    return a.score > b.score  -- 按分数降序
end)
for _, s in ipairs(scores) do
    print(s.name, s.score)
end

-- table.concat：远快于循环拼接！
local parts = {"Hello", " ", "World", "!", " "}
for i = 1, 10000 do
    table.insert(parts, tostring(i % 10))
end
local result = table.concat(parts)  -- 高效！
-- ❌ 慢速：result = ""; for _, p in ipairs(parts) do result = result .. p end

-- table.move（Lua 5.3+）
local a = {1, 2, 3, 4, 5}
table.move(a, 1, 3, 5, a)  -- 在原地移动：a = {1,2,3,4,1,2,3}

-- table.pack / unpack
local packed = table.pack(1, "two", true, nil)  -- 保留 nil
print(packed.n, packed[1])  --> 4, 1

-- ========== 2. String 库与模式匹配 ==========

-- 基本模式匹配
local text = "Contact: alice@example.com, bob@test.org"
for email in text:gmatch("[%w%.%-]+@[%w%.%-]+%.%w+") do
    print("Email found:", email)
end

-- 模式匹配示例
local date = "2026-05-14"
local year, month, day = date:match("(%d+)%-(%d+)%-(%d+)")
print(string.format("%s年%s月%s日", year, month, day))

-- gsub：替换
local template = "Hello, ${name}! Your score is ${score}."
local filled = template:gsub("%$%{(%w+)%}", {
    name = "Alice",
    score = "95",
})
print(filled)  --> "Hello, Alice! Your score is 95."

-- 常用字符串操作
local s = "  Hello World  "
print(s:upper())             --> "  HELLO WORLD  "
print(s:lower())             --> "  hello world  "
print(s:gsub("^%s+", ""))   --> "Hello World  "（去除前导空白）
print(s:match("^%s*(.-)%s*$"))  --> "Hello World"（去除首尾空白）
print(#s)                    --> 15（长度）
print(s:find("World"))      --> 8, 12（起始和结束位置）
print(s:sub(8, 12))         --> "World"（子串）

-- ========== 3. Math 库 ==========

-- 基本数学
print(math.sin(math.pi / 2))   --> 1.0
print(math.sqrt(144))          --> 12.0
print(math.max(3, 7, 2, 9))   --> 9

-- 随机数（Lua 5.4 改进）
math.randomseed(os.time())
print(math.random(1, 100))     -- 1-100 之间的随机整数
print(math.random())           -- [0, 1) 之间的随机浮点数

-- 整数运算（Lua 5.3+）
print(math.tointeger(3.14))    --> nil（不是整数）
print(math.tointeger(42.0))    --> 42
print(math.type(3.14))         --> "float"
print(math.type(42))           --> "integer"
print(math.ult(1, 2))          --> true（无符号小于比较）

-- ========== 4. IO 库 ==========

-- 写文件
local file = io.open("output.txt", "w")
if file then
    file:write("Line 1\n")
    file:write("Line 2\n")
    
    -- 格式化写入
    file:write(string.format("Value: %.2f\n", 3.14159))
    
    file:close()
end

-- 读文件
local file2 = io.open("output.txt", "r")
if file2 then
    -- 逐行读取
    for line in file2:lines() do
        print("Read:", line)
    end
    
    -- 重置到开头
    file2:seek("set")
    
    -- 全部读取
    local content = file2:read("*a")
    print("Full content:", content)
    
    file2:close()
end

-- ========== 5. OS 库 ==========

-- 时间操作
local now = os.time()
local dateStr = os.date("%Y-%m-%d %H:%M:%S", now)
print("现在时间:", dateStr)

-- 时间戳计算
local tomorrow = now + 24 * 60 * 60
print("明天:", os.date("%Y-%m-%d", tomorrow))

-- 从日期字符串构建时间戳
local someday = os.time({year=2026, month=12, day=25})
print("圣诞节是:", os.date("%A", someday))

-- 执行外部命令（谨慎使用）
-- local exitCode = os.execute("echo Hello from shell")

-- ========== 6. Debug 库（开发/调试神器） ==========

local function inspectFunction(fn)
    local info = debug.getinfo(fn, "S")
    print("Source:", info.source)
    print("Line defined:", info.linedefined)
    print("Last line:", info.lastlinedefined)
end

local function showTraceback()
    print(debug.traceback("当前调用栈:", 1))
end

local function level3()
    showTraceback()  -- 显示谁调用了 level3
end

local function level2()
    level3()
end

local function level1()
    level2()
end

level1()
-- 输出类似：
-- stack traceback:
--     test.lua:13: in function 'showTraceback'
--     test.lua:17: in function 'level3'
--     test.lua:21: in function 'level2'
--     test.lua:25: in function 'level1'
```

## 推荐书籍

1. **《Programming in Lua》（第4版）** — Roberto Ierusalimschy
   第 11 章到第 15 章完整覆盖了所有标准库，第 22 章专门讲字符串的模式匹配。

2. **《Lua 5.4 Reference Manual》**（在线免费）
   第 6 节（Standard Libraries）是每个函数的精确定义和边界行为。

3. **《Beginning Lua Programming》** — Kurt Jung / Aaron Brown
   以大量实用示例讲解标准库的常见应用场景，适合边练边学。
