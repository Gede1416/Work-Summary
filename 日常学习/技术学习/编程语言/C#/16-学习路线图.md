# C# 学习路线图

## 阶段一：基础入门（1-2 个月）

**目标**：能用 C# 写控制台程序，理解类型系统和 OOP 基础。

- 值类型 vs 引用类型、可空类型
- 类、继承、接口、抽象类
- 属性、索引器、事件
- 集合：List、Dictionary、LINQ 基础
- 异常处理、using 语句
- 文件 I/O、JSON 序列化

**里程碑**：实现一个命令行 TODO 应用（JSON 持久化存储）。

**推荐资源**：**《C# 12 in a Nutshell》第 1-4 章**。

## 阶段二：OOP 与 .NET 平台（2-3 个月）

**目标**：掌握 .NET 平台基础和面向对象设计。

- SOLID 原则在 C# 中的实践
- 泛型：泛型类/方法、协变/逆变
- LINQ 深入：方法语法、查询语法、延迟执行
- 委托、事件、Lambda、闭包
- async/await 基础
- 依赖注入概念

**里程碑**：实现一个简单的订单管理系统（分层架构）。

**推荐资源**：**《C# 12 in a Nutshell》第 5-15 章**，**《Adaptive Code via C#》**。

## 阶段三：ASP.NET Core Web 开发（3-4 个月）

**目标**：能用 ASP.NET Core 构建生产级 Web API。

- Minimal API 与 Controller-based API
- Entity Framework Core（Code First、Migration）
- 认证与授权（JWT、OAuth2）
- 中间件管道
- 日志（Serilog）、配置管理
- 单元测试（xUnit + Moq）
- Swagger/OpenAPI 文档

**里程碑**：实现一个完整的 REST API（用户管理 + CRUD + 认证）。

**推荐资源**：**《ASP.NET Core in Action》**，**《EF Core in Action》**。

## 阶段四：高级主题与性能（2-3 个月）

**目标**：掌握 .NET 高级特性和性能优化。

- Span\<T\>/Memory\<T\> 零分配编程
- Channel、IAsyncEnumerable、高级异步模式
- 反射与 Source Generator
- GC 深入：分代、LOH、POH、GC 模式
- BenchmarkDotNet 基准测试
- Docker 容器化部署

**里程碑**：优化一个现有 API 达到 < 1ms 的 p95 延迟。

**推荐资源**：**《CLR via C#》**，**《Pro .NET Memory Management》**。

## 阶段五：架构与分布式（3-6 个月）

**目标**：具备设计和实现分布式系统的能力。

- 微服务架构与 API 网关
- 消息队列（RabbitMQ + MassTransit）
- gRPC 服务间通信
- SignalR 实时通信
- CQRS 与事件溯源
- 分布式追踪（OpenTelemetry）

**里程碑**：实现一个微服务电商系统（订单服务 + 库存服务 + 支付服务）。

**推荐资源**：**微软 Architecture eBooks**，**《Building Microservices》**。
