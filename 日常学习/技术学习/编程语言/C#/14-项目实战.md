# C# 项目实战

## 详细描述

C# 的项目实战侧重于理解托管运行时和框架设计的优势——如何最大化利用 .NET 平台的抽象能力构建可维护的应用。

**Web API 项目**是 C# 最常见的实战场景。从零构建一个 ASP.NET Core REST API：定义实体（Domain Models）、仓储接口（Repository Pattern）、服务层（Business Logic）、控制器（Endpoints）、中间件（认证、日志、异常处理）。使用 EF Core 管理数据访问，FluentValidation 验证输入。

**微服务架构**：将 Web API 拆分为独立的服务，引入 API 网关（YARP/Ocelot）、服务发现（Consul）、消息队列（RabbitMQ + MassTransit）、分布式追踪（OpenTelemetry）。

**桌面应用**（WPF/MAUI）：使用 MVVM 模式构建桌面 GUI。ViewModel 实现 `INotifyPropertyChanged`，通过数据绑定与 View 同步。依赖注入容器管理 ViewModel 生命周期。

**Unity 游戏脚本**：Unity 使用 C# 作为核心脚本语言。GameObject + Component 模式，MonoBehaviour 生命周期方法（`Start`、`Update`、`FixedUpdate`、`OnCollisionEnter`），ScriptableObject 共享数据配置。

## 代码实例

```csharp
// ========== ASP.NET Core Web API 完整示例 ==========
// Program.cs
var builder = WebApplication.CreateBuilder(args);
builder.Services.AddDbContext<AppDbContext>(o =>
    o.UseSqlServer(builder.Configuration.GetConnectionString("Default")));
builder.Services.AddScoped<IOrderRepository, OrderRepository>();
builder.Services.AddScoped<IOrderService, OrderService>();
builder.Services.AddValidatorsFromAssemblyContaining<CreateOrderValidator>();
builder.Services.AddEndpointsApiExplorer();
builder.Services.AddSwaggerGen();

var app = builder.Build();
app.UseMiddleware<GlobalExceptionMiddleware>();
app.UseSwagger();
app.UseSwaggerUI();

// Minimal API 端点
app.MapGet("/orders/{id}", async (int id, IOrderService service, CancellationToken ct) =>
{
    var order = await service.GetByIdAsync(id, ct);
    return order is not null ? Results.Ok(order) : Results.NotFound();
});

app.MapPost("/orders", async (CreateOrderRequest request,
    IOrderService service, IValidator<CreateOrderRequest> validator, CancellationToken ct) =>
{
    var validation = await validator.ValidateAsync(request, ct);
    if (!validation.IsValid) return Results.ValidationProblem(validation.ToDictionary());
    
    var order = await service.CreateAsync(request, ct);
    return Results.Created($"/orders/{order.Id}", order);
});

app.Run();

// 实体
public class Order
{
    public int Id { get; private set; }
    public string CustomerName { get; private set; } = "";
    public List<OrderItem> Items { get; private set; } = new();
    public OrderStatus Status { get; private set; } = OrderStatus.Pending;
    
    public void AddItem(string product, int quantity, decimal price)
    {
        Items.Add(new OrderItem { ProductName = product, Quantity = quantity, Price = price });
    }
    
    public void Process() => Status = OrderStatus.Processed;
}

// DDD 值对象
public record Money(decimal Amount, string Currency);
public record Address(string Street, string City, string Country, string PostCode);
```
