# C# 函数与作用域

## 详细描述

C# 的函数体系随着语言版本演进不断丰富，从经典的静态方法到现代的表达式体成员、局部函数和 lambda 闭包，为不同场景提供了恰当的抽象手段。

**方法**是 C# 中最基本的函数形式，分为实例方法和静态方法。实例方法通过对象调用，可以访问实例状态；静态方法属于类型本身。方法支持默认参数和命名参数，调用时可以通过参数名指定实参，不必按位置顺序。

**表达式体方法**（C# 6/7）允许用 `=>` 语法简化单表达式方法的写法，不仅限于方法和属性，还包括构造函数、析构函数、索引器等。这种语法让代码更简洁、更函数式。

**局部函数**（C# 7.0）是在方法内部定义的函数，可以访问外层方法的变量和参数。与 lambda 相比，局部函数有几个优势：支持 `ref`/`out` 参数、支持 `yield return`、可以定义在代码的任何位置（甚至在使用之后）、编译器可以进行尾递归优化。局部函数在迭代器方法中特别有用——可以在局部函数中进行参数验证，而不会延迟到首次迭代。

**Lambda 表达式**分为语句 lambda（`(x) => { return x * x; }`）和表达式 lambda（`(x) => x * x`）。C# 10 支持了 lambda 的自然类型推断，可以直接用 `var` 声明 lambda 变量。Lambda 可以捕获外层变量形成闭包——编译器将捕获的变量提升为堆上的对象字段。

**闭包**是 C# 中需要理解的关键概念。当 lambda 或匿名函数引用外层变量时，编译器生成一个闭包类来持有这些变量。这意味着被捕获变量的生命周期会延长，在 foreach 循环中使用 lambda 捕获循环变量是经典的陷阱（C# 5 已修复 foreach 的行为）。

**命名空间与 using 指令**：命名空间组织代码，`using` 可以导入命名空间或为类型创建别名。C# 10 引入的**文件作用域命名空间**（`namespace MyApp;`）减少了一层缩进。**全局 using**（C# 10）允许在一个文件中声明全局生效的 using，配合 `<ImplicitUsings>` 项目设置自动导入常用命名空间。

**ref 参数与返回值**：`ref`、`out`、`in` 修饰符提供对变量的引用传递。`ref` 要求初始化后传入并可修改；`out` 不需要初始化但必须在方法内赋值；`in` 传递只读引用。C# 7 引入了 ref 返回值和 ref 局部变量，让高性能场景可以在安全代码中避免拷贝。

// ========== 1. 方法、默认参数与命名参数 ==========
public class Calculator
{
    // 表达式体方法（C# 6+）
    public static int Add(int a, int b) => a + b;
    
    // 默认参数 + 命名参数
    public static double CalculateDiscount(
        double price,
        double discountRate = 0.1,
        bool isMember = false)
    {
        double rate = isMember ? discountRate + 0.05 : discountRate;
        return price * (1 - rate);
    }
    
    // ref 参数
    public static void Swap<T>(ref T a, ref T b)
    {
        (a, b) = (b, a);
    }
    
    // out 参数
    public static bool TryDivide(int a, int b, out int result)
    {
        if (b == 0) { result = 0; return false; }
        result = a / b;
        return true;
    }
    
    // in 参数（只读引用，避免大结构体拷贝）
    public static double Distance(in Point3D p1, in Point3D p2)
    {
        double dx = p1.X - p2.X;
        double dy = p1.Y - p2.Y;
        double dz = p1.Z - p2.Z;
        return Math.Sqrt(dx * dx + dy * dy + dz * dz);
    }
}

// ========== 2. 局部函数 ==========
public class IteratorDemo
{
    // 局部函数在迭代器中的应用
    public static IEnumerable<int> FilterAndTransform(
        IEnumerable<int> source, Func<int, bool> predicate)
    {
        ArgumentNullException.ThrowIfNull(source);
        ArgumentNullException.ThrowIfNull(predicate);
        
        // 局部函数：包含 yield return
        return Impl();
        
        IEnumerable<int> Impl()
        {
            foreach (var item in source)
                if (predicate(item))
                    yield return item * item;
        }
    }
    
    // 局部函数做尾递归
    public static int Factorial(int n)
    {
        if (n < 0) throw new ArgumentOutOfRangeException(nameof(n));
        
        return Compute(n, 1);
        
        static int Compute(int n, int acc) =>
            n <= 1 ? acc : Compute(n - 1, acc * n);
    }
}

// ========== 3. Lambda 与闭包 ==========
public class LambdaExamples
{
    public static void Demonstrate()
    {
        // 表达式 lambda
        Func<int, int> square = x => x * x;
        
        // 语句 lambda
        Action<string> log = message =>
        {
            var timestamp = DateTime.Now.ToString("HH:mm:ss");
            Console.WriteLine($"[{timestamp}] {message}");
        };
        
        // C# 10：var 声明 lambda
        var cube = (int x) => x * x * x;
        var greet = () => "Hello, World!";
        
        // 闭包：捕获外层变量
        int counter = 0;
        Func<int> increment = () => ++counter;
        
        Console.WriteLine(increment()); // 1
        Console.WriteLine(increment()); // 2
        Console.WriteLine(counter);     // 2（共享同一变量）
        
        // 闭包在 LINQ 中的应用
        int threshold = 5;
        int[] numbers = { 1, 3, 5, 7, 9, 11 };
        var filtered = numbers.Where(n => n > threshold);  // 捕获 threshold
        
        // 注意：闭包捕获的是变量，不是值
        var actions = new List<Action>();
        for (int i = 0; i < 3; i++)
        {
            int captured = i;  // 关键：每次迭代拷贝
            actions.Add(() => Console.WriteLine(captured));
        }
        actions.ForEach(a => a()); // 0, 1, 2（正确！）
        
        // 如果直接用 i，会输出 3, 3, 3（经典陷阱）
    }
}

// ========== 5. 高阶函数 ==========
public static class HigherOrderFunctions
{
    // 函数作为参数
    public static T[] Filter<T>(T[] array, Func<T, bool> predicate)
    {
        return array.Where(predicate).ToArray();
    }
    
    // 函数作为返回值（柯里化）
    public static Func<int, int> MultiplyBy(int factor) =>
        x => x * factor;
    
    // 函数组合
    public static Func<T, TResult2> Compose<T, TResult1, TResult2>(
        Func<T, TResult1> f,
        Func<TResult1, TResult2> g) =>
        x => g(f(x));
    
    public static void Demonstrate()
    {
        var double_ = MultiplyBy(2);
        var triple = MultiplyBy(3);
        Console.WriteLine(double_(5));  // 10
        Console.WriteLine(triple(5));   // 15
        
        var addOne = (int x) => x + 1;
        var doubleIt = (int x) => x * 2;
        var addOneThenDouble = Compose(addOne, doubleIt);
        
        Console.WriteLine(addOneThenDouble(5)); // (5+1)*2 = 12
    }
}

## 推荐书籍

1. **《C# in Depth》（第4版）** — Jon Skeet
   详细讲解 C# 各版本的函数演进：lambda、闭包、局部函数、表达式体成员等，深入编译器生成的代码。

2. **《Functional Programming in C#》（第2版）** — Enrico Buonanno
   系统介绍如何在 C# 中使用函数式编程范式，包括高阶函数、函数组合、柯里化和不可变性。

3. **《C# 12 in a Nutshell》** — Joseph Albahari
   "Functions" 和 "Lambda Expressions" 章节提供了所有函数形式的完整参考和实战示例。
