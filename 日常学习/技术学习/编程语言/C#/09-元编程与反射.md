# C# 元编程与反射

## 详细描述

C# 拥有强大的运行时反射和编译时代码生成能力，这是 .NET 平台区别于 C++ 和 Rust 等系统语言的重要特性。

**反射**（Reflection）通过 `System.Reflection` 命名空间在运行时检查和操作类型信息：
- `Type.GetType()` 或 `typeof()` 获取类型对象
- `GetProperties()`、`GetMethods()`、`GetFields()` 获取成员
- `GetCustomAttributes()` 获取特性（Attribute）
- `Activator.CreateInstance()` 动态创建对象
- `MethodInfo.Invoke()` 动态调用方法

反射可以实现依赖注入容器、ORM、序列化库、插件系统、测试框架等高级功能。但反射有性能开销——MethodBase.Invoke 比直接调用慢 10-100 倍。

**特性**（Attribute）是编译期声明元数据的方式，通过反射在运行时读取。自定义特性继承 `System.Attribute`，用 `[MyAttribute]` 语法应用到类、方法、属性、参数等目标上。`[Required]`、`[JsonProperty]`、`[HttpGet]` 等框架特性本质都是通过反射 + 特性实现的。

**代码生成器**（Source Generator, .NET 5+）是元编程的未来方向——在编译过程中生成额外的 C# 源代码，集成到编译产物中。与反射不同，代码生成器是编译期的，零运行时开销。典型用例：序列化器（`System.Text.Json` 的源生成器）、依赖注入注册代码、INotifyPropertyChanged 实现。

**表达式树**（Expression Trees）将代码表示为可以分析、修改和编译的数据结构。`Expression<Func<int, int>>` 不是委托，而是一棵表示 lambda 表达式的语法树。LINQ to SQL / EF Core 将 C# 表达式转换为 SQL 正是通过表达式树实现的。

**dynamic 类型**绕过编译时类型检查，在运行时进行动态分派（通过 DLR，Dynamic Language Runtime）。主要用于 COM 互操作、与动态语言（IronPython、IronRuby）交互、处理无固定结构的数据（如 JSON 未知结构）。`ExpandoObject` 和 `DynamicObject` 可以自定义动态行为。

// ========== 1. 反射基础 ==========

public class ReflectionDemo
{
    public static void Inspect<T>()
    {
        Type type = typeof(T);
        Console.WriteLine($"Type: {type.FullName}");
        Console.WriteLine($"Base type: {type.BaseType}");
        Console.WriteLine($"Is class: {type.IsClass}");
        Console.WriteLine($"Is generic: {type.IsGenericType}");

        Console.WriteLine("\nProperties:");
        foreach (var prop in type.GetProperties())
        {
            Console.WriteLine($"  {prop.PropertyType.Name} {prop.Name} " +
                $"{{get={(prop.CanRead ? "✓" : "✗")}, set={(prop.CanWrite ? "✓" : "✗")}}}");
        }

        Console.WriteLine("\nMethods:");
        foreach (var method in type.GetMethods(
            BindingFlags.Public | BindingFlags.Instance | BindingFlags.DeclaredOnly))
        {
            var parameters = string.Join(", ",
                method.GetParameters().Select(p => $"{p.ParameterType.Name} {p.Name}"));
            Console.WriteLine($"  {method.ReturnType.Name} {method.Name}({parameters})");
        }
    }

    // 动态创建与调用
    public static object? CreateAndInvoke(string typeName, string methodName, params object[] args)
    {
        Type? type = Type.GetType(typeName);
        if (type is null) return null;

        object? instance = Activator.CreateInstance(type);
        var method = type.GetMethod(methodName);
        return method?.Invoke(instance, args);
    }
}

// ========== 2. 自定义特性 ==========

[AttributeUsage(AttributeTargets.Class | AttributeTargets.Property, AllowMultiple = false)]
public class TableAttribute : Attribute
{
    public string Name { get; }
    public TableAttribute(string name) => Name = name;
}

[AttributeUsage(AttributeTargets.Property)]
public class ColumnAttribute : Attribute
{
    public string Name { get; }
    public bool Required { get; init; }
    public int MaxLength { get; init; } = 255;
    
    public ColumnAttribute(string name) => Name = name;
}

// 使用自定义特性
[Table("users")]
public class User
{
    [Column("id", Required = true)]
    public int Id { get; set; }

    [Column("user_name", Required = true, MaxLength = 100)]
    public string Name { get; set; } = "";

    [Column("email", MaxLength = 256)]
    public string? Email { get; set; }

    // 没有 Column 特性的属性不会映射到数据库
    public string DisplayName => $"{Name} <{Email}>";
}

// 利用特性生成 SQL（ORM 的核心原理）
public static class SqlGenerator
{
    public static string GenerateInsert<T>(T entity) where T : class
    {
        var type = typeof(T);
        var tableAttr = type.GetCustomAttribute<TableAttribute>();
        if (tableAttr is null) throw new InvalidOperationException("No table attribute found");

        var columns = new List<string>();
        var values = new List<string>();

        foreach (var prop in type.GetProperties())
        {
            var colAttr = prop.GetCustomAttribute<ColumnAttribute>();
            if (colAttr is null) continue;

            var value = prop.GetValue(entity);
            if (value is null && colAttr.Required)
                throw new InvalidOperationException($"Required column {colAttr.Name} is null");

            columns.Add(colAttr.Name);
            values.Add(value is string s ? $"'{s}'" : value?.ToString() ?? "NULL");
        }

        return $"INSERT INTO {tableAttr.Name} ({string.Join(", ", columns)}) " +
               $"VALUES ({string.Join(", ", values)})";
    }
}

// ========== 3. 表达式树 ==========

public static class ExpressionTreeDemo
{
    public static void Demonstrate()
    {
        // 表达式树：不是委托，是可分析的代码结构
        Expression<Func<int, int, int>> expr = (a, b) => a + b;

        // 分析表达式结构
        var body = (BinaryExpression)expr.Body;
        Console.WriteLine($"Operation: {body.NodeType}");        // Add
        Console.WriteLine($"Left: {((ParameterExpression)body.Left).Name}");   // a
        Console.WriteLine($"Right: {((ParameterExpression)body.Right).Name}"); // b

        // 编译为委托并执行
        var compiled = expr.Compile();
        Console.WriteLine($"Result: {compiled(3, 4)}");  // 7

        // 动态构建表达式树
        var paramX = Expression.Parameter(typeof(int), "x");
        var constant2 = Expression.Constant(2);
        var multiply = Expression.Multiply(paramX, constant2);
        var constant1 = Expression.Constant(1);
        var add = Expression.Add(multiply, constant1);
        // 等价于：x => x * 2 + 1

        var lambda = Expression.Lambda<Func<int, int>>(add, paramX);
        var func = lambda.Compile();
        Console.WriteLine($"f(5) = {func(5)}");  // 11
    }
}

// ========== 5. dynamic 动态类型 ==========

public static class DynamicDemo
{
    public static void Demonstrate()
    {
        // ExpandoObject：动态添加属性
        dynamic person = new System.Dynamic.ExpandoObject();
        person.Name = "Alice";
        person.Age = 30;
        person.Greet = (Func<string>)(() => $"Hello, I'm {person.Name}");

        Console.WriteLine(person.Name);     // "Alice"
        Console.WriteLine(person.Greet());  // "Hello, I'm Alice"

        // 动态 JSON 处理（避免定义类）
        string json = @"{""title"": ""C# Book"", ""price"": 49.99}";
        dynamic? book = JsonSerializer.Deserialize<System.Dynamic.ExpandoObject>(json);

## 推荐书籍

1. **《C# 12 in a Nutshell》** — Joseph Albahari
   第 19 章（Reflection and Metadata）和第 20 章（Dynamic Programming）是 C# 反射和动态编程的完整参考。

2. **《CLR via C#》（第4版）** — Jeffrey Richter
   第 23 章（Assembly Loading and Reflection）从 CLR 内部讲解反射的工作机制和性能特征。

3. **《Metaprogramming in .NET》** — Kevin Hazzard / Jason Bock
   系统讲解 .NET 元编程：反射、表达式树、Emit、动态编程、Roslyn 分析和代码生成。
