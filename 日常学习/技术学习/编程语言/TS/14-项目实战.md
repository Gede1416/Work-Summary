# TypeScript 项目实战

## 详细描述

TypeScript 的实战能力通过全栈项目、类型挑战和工具链定制三个维度来检验。

**全栈项目**（Next.js + tRPC + Prisma）是验证 TS 全栈能力的最佳组合。前端使用 Next.js App Router + React Server Components，通过 tRPC 定义类型安全的 API 契约，Prisma 管理数据库 Schema 并生成 TS 类型。整个项目从数据库到 UI 的端到端类型安全，一个字段名改了编译器立即告诉你所有影响点。

**类型挑战**（Type Challenges）是 TS 类型体操的终极训练。通过实现 `DeepReadonly`、`TupleToUnion`、`StringToUnion`、`Chainable Options` 等类型操作，深入理解条件类型、映射类型、递归类型、模板字面量的组合用法。

**声明文件编写**是为 JS 库贡献类型定义的核心技能。理解 `declare`、`namespace`、`module` 的声明语法，为 CommonJS / ES Module / UMD 不同模块格式编写对应的 `.d.ts` 文件。

**工具链定制**：配置 monorepo（pnpm workspaces + Turborepo）、ESLint 规则链、Prettier 格式化、Git hooks（husky + lint-staged）、GitHub Actions CI/CD。

**TypeScript 编译器 API 应用**：使用 Compiler API 编写代码转换工具、自动化重构脚本或自定义 lint 规则。

## 代码实例

```typescript
// ========== 类型挑战示例：实现 DeepReadonly ==========
type DeepReadonly<T> = {
    readonly [K in keyof T]: T[K] extends object
        ? T[K] extends Function ? T[K] : DeepReadonly<T[K]>
        : T[K];
};

// ========== 声明文件结构 ==========
// my-library.d.ts
declare module "my-library" {
    export interface Config {
        apiKey: string;
        timeout?: number;
    }
    export function initialize(config: Config): void;
    export function query<T = unknown>(endpoint: string): Promise<T>;
}

// 扩展全局类型
declare global {
    interface Array<T> {
        last(): T | undefined;
        first(): T | undefined;
    }
}

// ========== tRPC + Prisma 端到端类型安全实战 ==========
// -- server/router.ts --
import { z } from "zod";
import { router, publicProcedure } from "./trpc";
import { prisma } from "./db";

export const appRouter = router({
    posts: router({
        list: publicProcedure
            .input(z.object({
                limit: z.number().min(1).max(100).default(10),
                cursor: z.string().optional(),
            }))
            .query(async ({ input }) => {
                const posts = await prisma.post.findMany({
                    take: input.limit + 1,
                    cursor: input.cursor ? { id: input.cursor } : undefined,
                    orderBy: { createdAt: "desc" },
                    include: { author: { select: { name: true } } },
                });
                // 类型自动推断包含 author.name
                return {
                    items: posts.slice(0, input.limit),
                    nextCursor: posts.length > input.limit ? posts[posts.length - 1]?.id : null,
                };
            }),
        create: publicProcedure
            .input(z.object({ title: z.string().min(1), content: z.string() }))
            .mutation(({ input, ctx }) => {
                return prisma.post.create({
                    data: {
                        title: input.title,
                        content: input.content,
                        authorId: ctx.userId,
                    },
                });
            }),
    }),
});

export type AppRouter = typeof appRouter;
// 客户端自动获得完整类型！无需任何代码生成
```

## 推荐书籍

1. **《Full Stack TypeScript》** — Nate Murray（在线免费）
   覆盖 Next.js + tRPC + Prisma + Tailwind 的全栈实战，每个部分都有可运行的完整项目。
2. **Type Challenges**（github.com/type-challenges）
   由浅入深的 100+ 道 TypeScript 类型体操题，是提升 TS 类型编程能力的最高效方式。
3. **《Total TypeScript》** — Matt Pocock（在线课程+书）
   从基础到极致的 TS 教程，包含声明文件编写、泛型模式、Compiler API 等高级主题。
