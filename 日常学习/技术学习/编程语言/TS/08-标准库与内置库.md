# TypeScript 标准库与内置库

## 详细描述

TypeScript 的"标准库"包含两个层面：ECMAScript 标准库（浏览器和 Node.js 共有）和运行时专用 API（Node.js API 或 Web API）。理解这些内置能力是高效 TS 开发的起点。

**ES 内置对象**是 TS 最基础的运行时支持：

- **Array**：`map`、`filter`、`reduce`、`find`、`some`、`every`、`flat`、`flatMap`、`sort`、`includes`、`at`（ES2022）。`reduce` 是 TS 中最强大的数组方法，可以实现 map、filter、groupBy 等几乎所有其他操作。

- **Map vs Object**：`Map` 保留插入顺序（与 Object 的字段顺序不可靠不同），键可以是任意类型（包括对象），有 `size` 属性和 `forEach` 方法。性能方面，频繁添加删除时 Map 更优，静态查询时 Object 更快。

- **Set**：唯一值集合，支持 `has`、`add`、`delete`、并集/交集/差集（通过迭代器）。`WeakSet` 只存储对象的弱引用。

- **Promise**：异步操作的标准抽象。`Promise.resolve`/`reject`、`.then`/`.catch`/`.finally`、`Promise.all`/`allSettled`/`any`/`race`。

- **Reflect**：元编程 API，统一了对象操作的低级方法：`Reflect.get`、`Reflect.set`、`Reflect.has`、`Reflect.deleteProperty`。与 Proxy 配合使用形成完整的拦截-转发模式。

- **Proxy**：拦截并自定义对目标对象的基本操作（get、set、has、delete、apply、construct 等），是实现响应式框架（Vue 3 的 Reactivity API 基于 Proxy）、数据验证、日志等的核心工具。

**Node.js API**（服务端 TS）提供了文件系统（`fs/promises`）、HTTP（`http`/`https`）、流（`stream`）、缓冲区（`Buffer`）、路径（`path`）、加密（`crypto`）等模块。

**Web API**（浏览器端 TS）包括 DOM API、`fetch`、WebSocket、`localStorage`、Canvas、Web Audio、IndexedDB、Service Worker 等。

**TypeScript 类型声明**是标准库的重要组成部分——TS 编译器自带 `lib` 类型定义（`lib.es2022.d.ts`、`lib.dom.d.ts` 等），为所有 JS/浏览器 API 提供类型注解。

## 代码实例

```typescript
// ========== 1. Array 高级操作 ==========

// reduce：万能的数组变换
const orders = [
    { id: 1, customer: "Alice", amount: 120, status: "completed" as const },
    { id: 2, customer: "Bob", amount: 85, status: "pending" as const },
    { id: 3, customer: "Alice", amount: 200, status: "completed" as const },
    { id: 4, customer: "Charlie", amount: 50, status: "cancelled" as const },
];

// groupBy 用 reduce
const byCustomer = orders.reduce<Record<string, typeof orders>>((acc, order) => {
    (acc[order.customer] ??= []).push(order);
    return acc;
}, {});

// 复杂聚合
const summary = orders.reduce(
    (acc, order) => {
        acc.totalAmount += order.amount;
        acc.countByStatus[order.status] = (acc.countByStatus[order.status] ?? 0) + 1;
        return acc;
    },
    {
        totalAmount: 0,
        countByStatus: {} as Record<string, number>,
    },
);

// flatMap：映射 + 展平
const tags = orders.flatMap(order => {
    const result: string[] = [];
    if (order.amount > 100) result.push("high-value");
    if (order.status === "pending") result.push("needs-attention");
    return result;
});
// ["high-value", "needs-attention", "high-value"]

// ========== 2. Map 与 Set ==========

// Map：对象作为键
const userCache = new Map<string, { name: string; lastAccess: Date }>();

userCache.set("alice", { name: "Alice", lastAccess: new Date() });

// Map 的迭代
for (const [key, value] of userCache) {
    console.log(`${key}: ${value.name}`);
}

// Set 运算
const setA = new Set([1, 2, 3, 4]);
const setB = new Set([3, 4, 5, 6]);

const union = new Set([...setA, ...setB]);          // {1,2,3,4,5,6}
const intersection = new Set([...setA].filter(x => setB.has(x)));  // {3,4}
const difference = new Set([...setA].filter(x => !setB.has(x)));   // {1,2}

// WeakMap：DOM 元素关联数据（不阻止 GC 的元素回收）
const elementData = new WeakMap<HTMLElement, { clickCount: number }>();

// ========== 3. Proxy 与 Reflect ==========

// 响应式对象
function reactive<T extends object>(target: T): T {
    return new Proxy(target, {
        get(obj, prop, receiver) {
            console.log(`[GET] ${String(prop)}`);
            // track(prop);  // 依赖追踪（如 Vue 3）
            return Reflect.get(obj, prop, receiver);
        },
        set(obj, prop, value, receiver) {
            const oldValue = Reflect.get(obj, prop, receiver);
            const result = Reflect.set(obj, prop, value, receiver);
            if (oldValue !== value) {
                console.log(`[SET] ${String(prop)}: ${oldValue} → ${value}`);
                // trigger(prop);  // 触发更新
            }
            return result;
        },
    });
}

const state = reactive({ count: 0, name: "Alice" });
state.count++;            // [GET] count → [SET] count: 0 → 1
console.log(state.name);  // [GET] name

// 验证代理
function validated<T extends Record<string, unknown>>(
    target: T,
    schema: { [K in keyof T]?: (v: unknown) => boolean },
): T {
    return new Proxy(target, {
        set(obj, prop, value) {
            const validator = (schema as any)[prop];
            if (validator && !validator(value)) {
                throw new Error(`Invalid value for ${String(prop)}: ${value}`);
            }
            return Reflect.set(obj, prop, value);
        },
    });
}

const user = validated({ age: 0, email: "" }, {
    age: (v) => typeof v === "number" && v >= 0,
    email: (v) => typeof v === "string" && v.includes("@"),
});
user.age = 30;   // OK
// user.age = -1;  // 抛出错误

// ========== 5. Web API（浏览器） ==========

// fetch + AbortController
async function fetchWithTimeout(url: string, timeoutMs: number): Promise<Response> {
    const controller = new AbortController();
    const timeoutId = setTimeout(() => controller.abort(), timeoutMs);
    try {
        const response = await fetch(url, { signal: controller.signal });
        return response;
    } finally {
        clearTimeout(timeoutId);
    }
}
```

## 推荐书籍

1. **《JavaScript: The Definitive Guide》（第7版）** — David Flanagan
   JS 百科全书，覆盖 ES2020 之前的所有标准库，Array/Map/Set/Promise/Proxy/Reflect 的详尽文档。

2. **《Node.js Design Patterns》（第3版）** — Mario Casciaro / Luciano Mammino
   系统覆盖 Node.js 标准库的实战应用：Stream、Buffer、EventEmitter、fs、http 等模块。

3. **《MDN Web Docs》**（在线资源，免费）
   浏览器端 API 的最权威参考——DOM、Canvas、WebSocket、Web Storage、Service Worker 等。
