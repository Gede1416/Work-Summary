# TypeScript 学习路线图

## 阶段一：TypeScript 类型基础（3-4 周）

**目标**：理解 TS 核心类型系统，能在 JS 项目中启用 TS。

- 基础类型：string/number/boolean/array/object
- 联合类型与交叉类型
- 接口（interface）与类型别名（type）
- 函数类型与参数类型
- 泛型基础：泛型函数、泛型约束
- tsconfig.json 基本配置

**里程碑**：将一个纯 JS 小项目迁移到 TS。

**推荐资源**：**《Programming TypeScript》第 1-5 章**，**《Learning TypeScript》**。

## 阶段二：高级类型与模式（3-4 周）

**目标**：掌握 TS 高级类型运算和设计模式。

- 条件类型与 infer
- 映射类型（Partial/Required/Readonly/Pick/Omit）
- 模板字面量类型
- Discriminated Union 领域建模
- unknown vs any、类型守卫
- 装饰器基础

**里程碑**：为解决一个类型挑战（Type Challenges Medium 难度）编写 5 个复杂类型工具。

**推荐资源**：**《Programming TypeScript》第 6-7 章**，**Type Challenges GitHub**。

## 阶段三：全栈框架实战（4-6 周）

**目标**：掌握一个主流 TS 全栈框架的完整开发流程。

- Next.js App Router + React Server Components
- tRPC（端到端类型安全 API）
- Prisma ORM（类型安全的数据库访问）
- Zod 数据验证
- Tailwind CSS 样式

**里程碑**：实现一个带用户认证的博客系统（文章 CRUD + 评论）。

**推荐资源**：**Next.js 官方教程**，**tRPC 官方文档**，**Prisma 官方文档**。

## 阶段四：Node.js 高级与性能（3-4 周）

**目标**：掌握 Node.js 高级模式和性能优化。

- Stream API 与背压（Backpressure）
- Worker Threads 并行计算
- 缓存策略（Redis 集成）
- 事件循环优化
- Node.js 调试与性能分析（clinic、0x）

**里程碑**：实现一个支持百万并发的 WebSocket 连接服务器。

**推荐资源**：**《Node.js Design Patterns》**，**《Effective TypeScript》第 40-62 条**。

## 阶段五：架构与工程化（4-6 周）

**目标**：具备设计和实施大型 TS 项目架构的能力。

- Monorepo 管理（pnpm workspaces / Turborepo）
- 模块联邦（Module Federation）微前端
- NestJS 企业级后端架构
- CI/CD 流水线（GitHub Actions）
- OpenTelemetry 分布式追踪

**里程碑**：搭建一个 monorepo 模板仓库，包含共享包、前端 App、后端 API。

**推荐资源**：**NestJS 官方文档**，**Turborepo 官方文档**。
