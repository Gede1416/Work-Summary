# TypeScript 版本演进

## 详细描述

TypeScript 从 2012 年发布至今，以每年 4 个主要版本的速度快速迭代。了解关键版本的里程碑特性有助于理解 TS 的设计哲学和演进方向。

**TS 1.x**（2014-2015）：基础类型系统建立。联合类型、交叉类型、类型守卫、泛型。

**TS 2.x**（2016-2018）：类型系统成熟。`strictNullChecks`（2.0，最重要的 TS 特性之一）、`keyof`/`typeof`（2.1）、条件类型（2.8）、`--strict` 选项（2.3）、Mapped Types。

**TS 3.x**（2018-2020）：高级类型操作。`unknown` 类型（3.0）、`const` 断言（3.4）、`as const`、可选链和空值合并（3.7）、`asserts` 类型谓词（3.7）、`import type`（3.8）。

**TS 4.x**（2020-2023）：类型系统增强。可变元组类型、模板字面量类型（4.1）、`infer` 增强、`noUncheckedIndexedAccess`（4.1）、`useUnknownInCatchVariables`（4.4）、`satisfies`（4.9）。

**TS 5.x**（2023-2026）：现代化和性能。装饰器标准化（5.0，ECMAScript Stage 3）、`const` 类型参数（5.0）、`export type *`（5.0）、`moduleResolution: bundler`（5.0）、`isolatedDeclarations`（5.5）。

**迁移策略**：从 JS 迁移到 TS 时，先启用 `allowJs`，从 `.js` 逐步重命名为 `.ts`，先 `noImplicitAny: false` 让项目能编译，再逐个文件夹启用 `strict` 并修复类型错误。

## 推荐书籍

1. **《Programming TypeScript》** — Boris Cherny，覆盖了到 TS 3.x 的核心特性。
2. **TypeScript 官方 Release Notes**（typescriptlang.org），每个版本的新特性详解和迁移指南。
3. **《Learning TypeScript》** — Josh Goldberg，覆盖到 TS 5.x 的最新特性。
