# C++ 基础语法与类型系统

## 详细描述

C++ 的类型系统是静态强类型系统，在编译期进行类型检查，兼具底层控制力和高层抽象能力。理解 C++ 类型系统是掌握这门语言的第一步。

**指针与引用**是 C++ 区别于大多数高级语言的核心特性。指针（`T*`）存储对象的内存地址，支持算术运算和空值（`nullptr`）；引用（`T&`）是对象的别名，一旦绑定便不能更改，语法上更安全。现代 C++ 推荐优先使用引用，只在需要所有权转移或可选语义时使用指针。

**const 系统**是 C++ 类型安全的重要支柱。`const` 变量不可修改，`const` 成员函数不修改对象状态，`const` 引用可以绑定到临时对象从而延长其生命周期。配合 `constexpr` 和 `consteval`（C++20），可以在编译期求值，实现零运行时开销的常量计算。

**auto 类型推导**（C++11）极大简化了复杂类型的声明，特别是模板迭代器和 lambda 表达式。配合 `decltype` 可以在不写出类型的情况下获取表达式的精确类型。C++14 引入了 `decltype(auto)` 进一步完善了返回值类型推导。

**移动语义**（C++11）是 C++ 性能优化的里程碑。通过右值引用（`T&&`）和 `std::move`，可以将资源的所有权从一个对象转移到另一个对象，避免深拷贝。配合移动构造函数和移动赋值运算符，可以显著提升容器和大型对象的性能。

**结构体、联合体与枚举**构成了 C++ 的复合类型体系。`struct` 默认公有继承，常用于 POD（Plain Old Data）类型；`union` 所有成员共享同一内存空间，C++17 引入了 `std::variant` 作为类型安全的替代；`enum class`（C++11）提供了有作用域的枚举，避免命名污染。

**现代 C++ 新特性**：C++17 的结构化绑定（`auto [x, y] = pair`）让多返回值语法更优雅；`if constexpr` 在编译期分支选择；C++20 的 `consteval` 保证函数必须在编译期执行；范围 for 循环简化了容器遍历。

## 代码实例

```cpp
#include <iostream>
#include <string>
#include <vector>
#include <memory>
#include <variant>
#include <concepts>

// ========== 1. 指针与引用 ==========
void pointer_and_reference() {
    int value = 42;
    
    // 引用：对象的别名，不能为 null
    int& ref = value;
    ref = 100;  // value 也变为 100
    
    // 指针：存储地址，可以为 nullptr
    int* ptr = &value;
    *ptr = 200;  // 通过指针修改
    
    // 空指针（C++11）
    int* null_ptr = nullptr;  // 替代 NULL 或 0
    
    // const 指针与指向 const 的指针
    const int* p1 = &value;   // 不能通过 p1 修改 value
    int* const p2 = &value;   // p2 本身不能指向别处
    const int* const p3 = &value; // 两者都不能修改
}

// ========== 2. constexpr 与 consteval ==========
constexpr int factorial(int n) {
    int result = 1;
    for (int i = 2; i <= n; ++i) result *= i;
    return result;
}
// factorial(5) 在编译期求值为 120

consteval int compile_time_square(int n) {
    return n * n;
}
// compile_time_square(10) 必须编译期求值，否则报错

// ========== 3. 移动语义 ==========
class Buffer {
    size_t size_;
    int* data_;
public:
    // 构造函数
    explicit Buffer(size_t size) : size_(size), data_(new int[size]) {}
    
    // 移动构造函数：窃取资源
    Buffer(Buffer&& other) noexcept 
        : size_(other.size_), data_(other.data_) {
        other.size_ = 0;
        other.data_ = nullptr;  // 将源对象置为安全状态
    }
    
    // 移动赋值运算符
    Buffer& operator=(Buffer&& other) noexcept {
        if (this != &other) {
            delete[] data_;
            size_ = other.size_;
            data_ = other.data_;
            other.size_ = 0;
            other.data_ = nullptr;
        }
        return *this;
    }
    
    ~Buffer() { delete[] data_; }
    
    // 禁止拷贝
    Buffer(const Buffer&) = delete;
    Buffer& operator=(const Buffer&) = delete;
};

// ========== 4. 结构化绑定 (C++17) ==========
struct Point { double x, y, z; };

void structured_bindings() {
    Point p{1.0, 2.0, 3.0};
    auto [x, y, z] = p;  // 结构化绑定
    
    std::map<std::string, int> ages{{"Alice", 30}, {"Bob", 25}};
    for (const auto& [name, age] : ages) {
        std::cout << name << " is " << age << "\n";
    }
}

// ========== 5. enum class 与 variant ==========
enum class Color { Red, Green, Blue };
enum class TrafficLight { Red, Yellow, Green };  // 不冲突

void enum_and_variant() {
    Color c = Color::Red;
    // int x = c;  // 错误！不能隐式转换
    int x = static_cast<int>(c);  // 正确，显式转换
    
    // std::variant：类型安全的联合体
    std::variant<int, double, std::string> v;
    v = 42;
    v = 3.14;
    v = "hello";
    
    // 访问 variant
    std::visit([](auto&& arg) {
        using T = std::decay_t<decltype(arg)>;
        if constexpr (std::is_same_v<T, int>)
            std::cout << "int: " << arg << "\n";
        else if constexpr (std::is_same_v<T, double>)
            std::cout << "double: " << arg << "\n";
        else
            std::cout << "string: " << arg << "\n";
    }, v);
}

// ========== 6. auto 与 decltype ==========
void type_deduction() {
    std::vector<int> vec{1, 2, 3, 4, 5};
    
    // auto 推导迭代器类型
    auto it = vec.begin();  // std::vector<int>::iterator
    
    // decltype 获取表达式类型
    decltype(vec)::value_type element = 10;  // int
    
    // decltype(auto) 保持引用和 cv 限定符
    const int& get_ref();
    decltype(auto) ref = get_ref();  // const int&
}

int main() {
    pointer_and_reference();
    structured_bindings();
    enum_and_variant();
    type_deduction();
    
    // 编译期求值
    constexpr int fact5 = factorial(5);  // 120
    constexpr int sq10 = compile_time_square(10);  // 100
    
    // 移动语义
    Buffer buf1(1024);
    Buffer buf2 = std::move(buf1);  // buf1 的资源移动到 buf2
    
    std::cout << "C++ 类型系统示例全部通过\n";
    return 0;
}
```

## 推荐书籍

1. **《C++ Primer》（第5版）** — Stanley B. Lippman / Josée Lajoie / Barbara E. Moo
   系统学习 C++ 基础语法和类型系统的最佳入门书，覆盖 C++11 标准，内容全面而深入，适合逐章精读。

2. **《Effective Modern C++》** — Scott Meyers
   深入讲解 C++11/14 的类型推导（auto、decltype）、移动语义、智能指针等现代特性，以条款形式组织，每个条款解决一个具体问题。

3. **《A Tour of C++》（第3版）** — Bjarne Stroustrup
   C++ 之父写的快速概览，覆盖 C++20 特性，适合有一定编程基础的读者快速掌握 C++ 核心语法和类型系统。

4. **《C++17 - The Complete Guide》** — Nicolai M. Josuttis
   全面覆盖 C++17 所有新特性，包括结构化绑定、if constexpr、std::variant 等，适合从 C++11/14 升级的学习者。
