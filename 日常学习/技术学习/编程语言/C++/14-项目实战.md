# C++ 项目实战

## 详细描述

C++ 的"从零构建"能力是区分真正理解底层和只会调用 API 的关键。以下实战项目从底层逐渐向上，从内存分配器到网络库再到游戏引擎模块。

**自建内存分配器**：实现一个简单的内存池（Pool Allocator）和栈分配器，理解 `malloc/free` 的内部逻辑，这是理解 STL allocator 和 `std::pmr` 的基础。

**自建模板容器**：从零实现 `Vector<T>`、`HashMap<K,V>` 和 `UniquePtr<T>`。通过这个过程深入理解 RAII、移动语义、异常安全和模板实例化。

**简单数据库**：实现一个基于 B+ Tree 的 Key-Value 存储引擎，支持插入、查找、范围扫描。深入理解持久化（WAL）、页缓存、崩溃恢复等数据库核心概念。

**网络库**：使用 `epoll`（Linux）或 `IOCP`（Windows）实现异步事件循环，支持 TCP 连接管理和自定义协议。Socket 编程到高性能服务器的完整链路。

**游戏引擎模块**：选择一个最小可行的引擎模块实现——比如渲染器的场景管理（BSP Tree 或 Octree）、物理引擎的碰撞检测（AABB + GJK 算法）、或动画系统的骨骼动画。

## 代码实例

```cpp
// ========== 简易内存池 ==========
template <typename T, size_t BlockSize = 256>
class MemoryPool {
    union Slot {
        T element;
        Slot* next;
        Slot() {}   // 不调用 T 的构造函数
        ~Slot() {}  // 不调用 T 的析构函数
    };

    struct Block {
        std::unique_ptr<Slot[]> slots;
        Block() : slots(std::make_unique<Slot[]>(BlockSize)) {
            // 初始化空闲链表
            for (size_t i = 0; i < BlockSize - 1; ++i)
                slots[i].next = &slots[i + 1];
            slots[BlockSize - 1].next = nullptr;
        }
    };

    Slot* freeList_ = nullptr;
    std::vector<Block> blocks_;

public:
    T* allocate() {
        if (!freeList_) {
            blocks_.emplace_back();
            freeList_ = &blocks_.back().slots[0];
        }
        Slot* slot = freeList_;
        freeList_ = freeList_->next;
        return reinterpret_cast<T*>(slot);
    }

    void deallocate(T* ptr) {
        auto* slot = reinterpret_cast<Slot*>(ptr);
        slot->next = freeList_;
        freeList_ = slot;
    }

    template <typename... Args>
    T* create(Args&&... args) {
        T* ptr = allocate();
        new (ptr) T(std::forward<Args>(args)...);
        return ptr;
    }

    void destroy(T* ptr) {
        ptr->~T();
        deallocate(ptr);
    }
};

// ========== 简易 UniquePtr ==========
template <typename T>
class UniquePtr {
    T* ptr_ = nullptr;
public:
    UniquePtr() = default;
    explicit UniquePtr(T* p) : ptr_(p) {}
    ~UniquePtr() { delete ptr_; }

    UniquePtr(const UniquePtr&) = delete;
    UniquePtr& operator=(const UniquePtr&) = delete;

    UniquePtr(UniquePtr&& other) noexcept : ptr_(other.ptr_) {
        other.ptr_ = nullptr;
    }
    UniquePtr& operator=(UniquePtr&& other) noexcept {
        if (this != &other) { delete ptr_; ptr_ = other.ptr_; other.ptr_ = nullptr; }
        return *this;
    }

    T* operator->() const { return ptr_; }
    T& operator*() const { return *ptr_; }
    explicit operator bool() const { return ptr_ != nullptr; }
    T* get() const { return ptr_; }
    T* release() { T* p = ptr_; ptr_ = nullptr; return p; }
};
```

## 推荐书籍

1. **《C++ Crash Course》** — Josh Lospinoso
   以项目驱动的方式学习现代 C++，从 Boost 集成到序列化、网络通信，适合动手党。
2. **《Game Engine Architecture》（第3版）** — Jason Gregory
   从游戏引擎全局视角讲解内存管理、资源系统、渲染管线和并发设计。
3. **《Build Your Own Database From Scratch》** — James Smith
   用 C++ 实现 B+ Tree K-V 存储引擎的完整教程。
