# C++ 标准库（STL）

## 详细描述

C++ 标准库（通常称为 STL）是现代 C++ 的基石。它提供了经过精心设计和优化的容器、算法、迭代器、字符串处理、文件系统操作等组件，让开发者能用声明式的方式写出高效、正确的代码。

**容器**分为三大类：
- **序列容器**：`vector`（动态数组，首选容器）、`deque`（双端队列）、`list`（双向链表）、`forward_list`（单向链表）、`array`（固定大小数组，C++11）
- **关联容器**：`set/multiset`（有序集合）、`map/multimap`（有序映射），基于红黑树
- **无序关联容器**：`unordered_set/map`、`unordered_multiset/map`（基于哈希表），O(1) 平均查找

**算法**（`<algorithm>`、`<numeric>`）提供了 100+ 个通用算法：查找（`find`、`binary_search`）、排序（`sort`、`partial_sort`、`nth_element`）、修改（`copy`、`transform`、`replace`）、删除（`remove`、`unique`）、数值（`accumulate`、`inner_product`）。算法的设计依赖于迭代器抽象——同一算法可用于数组、vector、list、甚至自定义容器。

**迭代器**是容器和算法之间的桥梁。五类迭代器：输入/输出迭代器（单次遍历）、前向迭代器（多次遍历，只能向前）、双向迭代器（可向前向后）、随机访问迭代器（常数时间跳到任意位置）。C++20 引入了 `std::ranges`，将算法重新设计为可直接接受范围的版本（`std::ranges::sort(v)` 代替 `std::sort(v.begin(), v.end())`）。

**\<string\>** 提供了 `std::string`（`std::basic_string<char>`），支持 UTF-8。C++17 引入了 `std::string_view`——一个不拥有字符串所有权的轻量级视图，避免不必要的字符串拷贝。

**\<filesystem\>**（C++17）提供了跨平台的路径操作、目录遍历、文件状态查询。让 C++ 首次有了标准化的文件管理 API。

**\<regex\>**（C++11）支持 ECMAScript 风格的正则表达式，但注意其性能开销较大——解析正则表达式的 `std::regex` 构造函数较慢。

**\<format\>**（C++20）提供了类似 Python f-string 的现代字符串格式化，类型安全、可扩展、性能优于 `sprintf` 和 `stringstream`。

**\<chrono\>**（C++11，C++20 大幅扩展）提供了类型安全的时间单位（`seconds`、`milliseconds`、`nanoseconds`）、时钟（`system_clock`、`steady_clock`）、日历和时区支持。

## 代码实例

```cpp
#include <iostream>
#include <vector>
#include <map>
#include <set>
#include <algorithm>
#include <numeric>
#include <string>
#include <string_view>
#include <format>
#include <filesystem>
#include <chrono>
#include <ranges>

// ========== 1. 容器 ==========
void container_examples() {
    // vector：首选动态数组
    std::vector<int> v{5, 2, 8, 1, 9, 3};
    
    // map：有序键值对
    std::map<std::string, int> ages{{"Alice", 30}, {"Bob", 25}};
    ages["Charlie"] = 35;
    
    // 结构化绑定遍历 map
    for (const auto& [name, age] : ages) {
        std::cout << std::format("{} is {} years old\n", name, age);
    }
    
    // set：有序唯一集合
    std::set<int> unique{3, 1, 4, 1, 5, 9, 2, 6};
    for (int n : unique) std::cout << n << " ";  // 1 2 3 4 5 6 9
    
    // unordered_map：哈希表
    std::unordered_map<std::string, double> prices{
        {"apple", 0.99}, {"banana", 0.59}
    };
    std::cout << "\nApple: $" << prices.at("apple") << "\n";
}

// ========== 2. 算法 ==========
void algorithm_examples() {
    std::vector<int> v{5, 2, 8, 1, 9, 3, 1, 4};
    
    // 排序
    std::sort(v.begin(), v.end());
    // v: {1, 1, 2, 3, 4, 5, 8, 9}
    
    // 查找
    if (auto it = std::find(v.begin(), v.end(), 4); it != v.end()) {
        std::cout << "Found 4 at index " << std::distance(v.begin(), it) << "\n";
    }
    
    // 条件查询
    bool has_even = std::any_of(v.begin(), v.end(), [](int n) { return n % 2 == 0; });
    auto even_count = std::count_if(v.begin(), v.end(), [](int n) { return n % 2 == 0; });
    
    // 变换
    std::vector<int> squared;
    std::transform(v.begin(), v.end(), std::back_inserter(squared),
                   [](int n) { return n * n; });
    
    // 删除-擦除惯用法
    auto new_end = std::remove_if(v.begin(), v.end(), [](int n) { return n < 3; });
    v.erase(new_end, v.end());
    // v: {3, 4, 5, 8, 9}
    
    // 数值算法
    int sum = std::accumulate(v.begin(), v.end(), 0);
    int product = std::accumulate(v.begin(), v.end(), 1, std::multiplies<>{});
    std::cout << "Sum: " << sum << ", Product: " << product << "\n";
}

// ========== 3. ranges（C++20） ==========
void ranges_examples() {
    std::vector<int> numbers{5, 2, 8, 1, 9, 3, 7, 4, 6};
    
    // 管道语法：过滤 + 转换
    auto result = numbers
        | std::views::filter([](int n) { return n % 2 == 0; })
        | std::views::transform([](int n) { return n * 10; })
        | std::ranges::to<std::vector<int>>();
    // result: {20, 80, 40, 60}
    
    // 懒惰视图（零拷贝）
    auto first_five = numbers | std::views::take(5);
    for (int n : first_five) std::cout << n << " ";  // 5 2 8 1 9
    
    // 键视图
    std::map<std::string, int> m{{"a", 1}, {"b", 2}, {"c", 3}};
    for (const auto& key : m | std::views::keys) {
        std::cout << key << " ";
    }
}

// ========== 4. string 与 string_view ==========
void string_examples() {
    // string_view：不拷贝，零开销的子串
    std::string text = "Hello, World!";
    std::string_view view = text;
    
    std::string_view world = view.substr(7, 5);  // "World"（没有内存分配！）
    std::cout << world << "\n";
    
    // format（C++20）
    std::string formatted = std::format("Pi ≈ {:.2f}, answer = {}", 3.14159, 42);
    std::cout << formatted << "\n";  // "Pi ≈ 3.14, answer = 42"
    
    // starts_with / ends_with（C++20）
    if (text.starts_with("Hello")) std::cout << "Greeting detected\n";
    if (text.ends_with("!")) std::cout << "Exclamation detected\n";
}

// ========== 5. filesystem ==========
void filesystem_examples() {
    namespace fs = std::filesystem;
    fs::path current = fs::current_path();
    std::cout << "Current path: " << current << "\n";
    
    // 遍历目录
    for (const auto& entry : fs::directory_iterator(current)) {
        std::cout << (entry.is_directory() ? "[DIR] " : "[FILE] ")
                  << entry.path().filename().string() << "\n";
    }
    
    // 路径操作
    fs::path config = current / "config" / "app.json";
    std::cout << "Extension: " << config.extension() << "\n";
    std::cout << "Filename: " << config.filename() << "\n";
    
    // 检查文件存在
    if (fs::exists(config)) {
        std::cout << "Config file exists, size: "
                  << fs::file_size(config) << " bytes\n";
    }
}

// ========== 6. chrono 时间库 ==========
void chrono_examples() {
    using namespace std::chrono;
    
    // 类型安全的时间运算
    auto start = steady_clock::now();
    
    // 模拟一些工作
    std::this_thread::sleep_for(100ms);
    
    auto end = steady_clock::now();
    auto elapsed = duration_cast<microseconds>(end - start);
    std::cout << "Elapsed: " << elapsed.count() << " µs\n";
    
    // C++20 日历
    auto today = floor<days>(system_clock::now());
    std::cout << "Days since epoch: " << today.time_since_epoch().count() << "\n";
}

int main() {
    container_examples();
    algorithm_examples();
    ranges_examples();
    string_examples();
    filesystem_examples();
    chrono_examples();
    return 0;
}
```

## 推荐书籍

1. **《The C++ Standard Library》（第2版）** — Nicolai M. Josuttis
   STL 的百科全书，覆盖容器、算法、迭代器、字符串、正则、IO、并发等所有标准库组件，每个组件都有详尽的使用示例和性能分析。

2. **《C++17 - The Complete Guide》** — Nicolai M. Josuttis
   专讲 string_view、filesystem、parallel algorithms、optional/variant/any 等 C++17 新增库。

3. **《C++20 - The Complete Guide》** — Nicolai M. Josuttis
   专讲 ranges、format、calendar/timezone、span 等 C++20 新增库和 ranges 库。

4. **《Functional Programming in C++》** — Ivan Cukic
   深入讲解算法和 ranges 的函数式组合模式，如何用 STL 写出声明式的数据处理流水线。
