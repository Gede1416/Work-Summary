# C++ 版本演进

## 详细描述

C++ 从 C++11 进入"现代 C++"时代，之后每 3 年发布一个新标准。了解各版本的关键特性是写出与时俱进的 C++ 代码的前提。

**C++11**（2011）：现代 C++ 的奠基版本。核心特性：`auto`/`decltype`、nullptr、智能指针（`unique_ptr`/`shared_ptr`）、移动语义（右值引用）、lambda、`constexpr`、`std::thread`、range-for、`enum class`、统一初始化（`{}`）、变参模板。

**C++14**（2014）：增量改进。泛型 lambda（`auto` 参数）、`decltype(auto)`、`make_unique`、二进制字面量、`std::integer_sequence`。

**C++17**（2017）：大量实用特性。结构化绑定（`auto [a, b] = pair`）、`if constexpr`、折叠表达式、`string_view`、`optional`/`variant`/`any`、`filesystem`、`std::clamp`、内联变量、CTAD（类模板实参推导）、并行 STL 算法。

**C++20**（2020）：革命性版本。`concept`、`ranges`、协程（`co_await`/`co_yield`）、`std::format`、`std::span`、`consteval`/`constinit`、`<=>`（三方比较运算符）、日历和时区、`std::jthread`、`std::stop_token`。

**C++23**（2023）：`std::expected`、`std::flat_map`/`std::flat_set`、`std::mdspan`（多维数组视图）、`std::generator`、显式对象参数（deducing this）、`if consteval`、`import std`（标准库模块）。

**迁移策略**：从旧标准升级到新标准时，一次一个版本逐步迁移。优先启用新编译器的警告，逐步应用推荐的新写法（`nullptr` 替代 `NULL`、`auto` 替代冗长类型、range-for 替代传统 for、smart pointer 替代裸指针）。

## 推荐书籍

1. **《A Tour of C++》（第3版）** — Bjarne Stroustrup，覆盖 C++20。
2. **《C++17 - The Complete Guide》** + **《C++20 - The Complete Guide》** — Nicolai M. Josuttis，每个特性的详细介绍和示例。
