# C++ 泛型、模板与多态

## 详细描述

C++ 的泛型编程通过**模板**实现，这是 C++ 最强大也最复杂的特性之一。与 Java/C# 的泛型不同，C++ 模板是一种编译期代码生成机制，每个不同的模板参数组合都会生成独立的代码，真正做到了"零成本抽象"。

**函数模板**允许编写与类型无关的算法。编译器根据调用时的实参类型进行模板实参推导（Template Argument Deduction, TAD），自动实例化正确的版本。可以显式指定模板参数来覆盖推导，也可以用 `auto`（C++17）简化模板函数声明。

**类模板**是泛型容器和数据结构的基础。STL 中的 `vector<T>`、`map<K,V>` 都是类模板。成员函数在类外定义时需要带上模板参数列表。类模板实参推导（CTAD，C++17）允许从构造函数参数推导模板参数，不再需要 `make_pair`、`make_tuple` 等辅助函数。

**变参模板**（Variadic Templates，C++11）支持任意数量的模板参数。通过递归展开参数包（Parameter Pack）实现编译期循环。C++17 的折叠表达式（Fold Expressions）极大简化了变参模板中参数包的归约操作。

**concept**（C++20）是对模板参数进行约束的革命性特性。它允许为模板参数定义"概念"——一组语义要求的集合，如"可排序"、"可打印"、"整型"。constraint 检查在编译期完成，错误信息清晰直接，不再出现几十行的模板实例化错误。concept 可以约束类型、变量、甚至 `auto` 推导。

**SFINAE**（Substitution Failure Is Not An Error）是 C++ 模板元编程的一个核心规则：在模板参数替换过程中，如果一个候选函数/类的替换失败，编译器不会报错，而是从候选集中移除这个选项。配合 `std::enable_if`，SFINAE 曾是实现编译期条件选择的唯一手段——C++20 的 concept 和 `requires` 从句已经让大多数 SFINAE 代码成为历史。

**编译期多态**（CRTP + concept）：模板提供静态多态（编译期多态），比虚函数的运行时多态更高效——没有 vtable 查询，可以内联。CRTP（Curiously Recurring Template Pattern）是静态多态的经典实现：`template<typename T> class Base { ... }; class Derived : public Base<Derived> { ... };`。

## 代码实例

```cpp
#include <iostream>
#include <vector>
#include <concepts>
#include <type_traits>

// ========== 1. 函数模板与模板实参推导 ==========
template <typename T>
T max(T a, T b) {
    return (a > b) ? a : b;
}

// C++17：用 auto 简化模板函数
auto min(auto a, auto b) { return (a < b) ? a : b; }

// ========== 2. 类模板与 CTAD ==========
template <typename T>
class Stack {
    std::vector<T> elements_;
public:
    void push(T const& elem) { elements_.push_back(elem); }
    void push(T&& elem) { elements_.push_back(std::move(elem)); }
    T pop() {
        T elem = std::move(elements_.back());
        elements_.pop_back();
        return elem;
    }
    bool empty() const { return elements_.empty(); }
};

// CTAD 推导指引（C++17）
// Stack(const char*) -> Stack<std::string>;  // 将字符串字面量推导为 string

void template_examples() {
    std::cout << max(3, 7) << "\n";          // int 版本
    std::cout << max(3.14, 2.72) << "\n";    // double 版本
    std::cout << min(10, 20) << "\n";
    
    Stack<int> intStack;
    intStack.push(1);
    intStack.push(2);
    std::cout << intStack.pop() << "\n";
    
    // CTAD：不需要写 Stack<int>
    Stack doubleStack = Stack{1.0, 2.0, 3.0};  // Stack<double>
}

// ========== 3. 变参模板与折叠表达式 ==========

// 递归展开（C++11 传统方式）
template <typename T>
T sum_recursive(T v) {
    return v;
}

template <typename T, typename... Args>
T sum_recursive(T first, Args... rest) {
    return first + sum_recursive(rest...);
}

// 折叠表达式（C++17，推荐方式）
template <typename... Args>
auto sum_fold(Args... args) {
    return (args + ...);  // 一元右折叠 (arg1 + (arg2 + (arg3 + ...)))
}

// 逗号运算符折叠：逐个打印
template <typename... Args>
void print_all(Args&&... args) {
    (std::cout << ... << args) << "\n";  // 二元左折叠
}

// ========== 4. concept（C++20） ==========

// 定义一个概念：要求类型支持乘法
template <typename T>
concept Multipliable = requires(T a, T b) {
    { a * b } -> std::convertible_to<T>;
};

// 约束函数模板
template <typename T>
requires Multipliable<T> && std::integral<T>
T square(T x) { return x * x; }

// 使用 concept 作为约束的简写形式
auto add(Multipliable auto a, Multipliable auto b) {
    return a + b;
}

// 更复杂的 concept
template <typename T>
concept Sortable = requires(T& container) {
    typename T::value_type;
    { container.begin() } -> std::input_iterator;
    { container.end() } -> std::input_iterator;
    requires std::totally_ordered<typename T::value_type>;
};

template <Sortable Container>
void sort_container(Container& c) {
    std::sort(c.begin(), c.end());
}

// ========== 5. SFINAE vs concept ==========

// 旧式 SFINAE + enable_if
template <typename T>
typename std::enable_if<std::is_integral_v<T>, T>::type
old_style(T x) { return x; }

// C++20 concept（等价且更清晰）
template <std::integral T>
T new_style(T x) { return x; }

// ========== 6. CRTP 静态多态 ==========
template <typename Derived>
class Animal {
public:
    void speak() const {
        // 在编译期决定调用哪个子类的实现
        static_cast<const Derived*>(this)->speak_impl();
    }
};

class Dog : public Animal<Dog> {
public:
    void speak_impl() const { std::cout << "Woof!\n"; }
};

class Cat : public Animal<Cat> {
public:
    void speak_impl() const { std::cout << "Meow!\n"; }
};

template <typename T>
void let_speak(const Animal<T>& animal) {
    animal.speak();  // 编译期多态，零开销
}

// ========== 7. 模板特化与偏特化 ==========
template <typename T>
struct TypeInfo {
    static const char* name() { return "unknown"; }
};

// 全特化
template <>
struct TypeInfo<int> {
    static const char* name() { return "int"; }
};

template <>
struct TypeInfo<double> {
    static const char* name() { return "double"; }
};

// 偏特化：匹配指针类型
template <typename T>
struct TypeInfo<T*> {
    static const char* name() {
        static std::string n = std::string(TypeInfo<T>::name()) + "*";
        return n.c_str();
    }
};

int main() {
    template_examples();
    
    std::cout << sum_recursive(1, 2, 3, 4, 5) << "\n";  // 15
    std::cout << sum_fold(1, 2, 3, 4, 5) << "\n";       // 15
    print_all("Hello", ", ", "World", "!");
    
    std::cout << square(5) << "\n";   // 适用（int 是整型且可乘）
    // square(3.14);  // 错误：double 不是 integral
    
    Dog dog;
    Cat cat;
    let_speak(dog);  // "Woof!"
    let_speak(cat);  // "Meow!"
    
    std::cout << TypeInfo<int>::name() << "\n";   // "int"
    std::cout << TypeInfo<int*>::name() << "\n";  // "int*"
    
    return 0;
}
```

## 推荐书籍

1. **《C++ Templates: The Complete Guide》（第2版）** — David Vandevoorde / Nicolai Josuttis / Douglas Gregor
   模板的百科全书，从基础到高级模板元编程，覆盖 C++17 变参模板和折叠表达式，是学习 C++ 泛型编程的必备参考。

2. **《C++20 - The Complete Guide》** — Nicolai M. Josuttis
   专门有一章详细讲解 concept 和 requires，从设计动机到实际使用，是所有升级 C++20 的开发者的必读。

3. **《Template Metaprogramming with C++》** — Marius Bancila
   全面讲解模板元编程实践，从 SFINAE 到 concept 的迁移，适合进阶学习者。
