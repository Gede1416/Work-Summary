# Lua 学习路线图

## 阶段一：基础入门（2-3 周）

**目标**：理解 Lua 的基本语法和独特之处，能写简单的脚本。

- 基础类型（nil/boolean/number/string/table）
- 多重赋值、local 变量、作用域
- table：数组和字典的统一数据结构（牢记索引从 1 开始）
- 函数定义、多返回值、闭包
- 控制流（if/for/while/repeat）
- 字符串模式匹配基础

**里程碑**：实现一个命令行数据处理脚本（读取文件、解析数据、输出报表）。

**推荐资源**：**《Programming in Lua》第 1-5 章**。

## 阶段二：Table 与元表深度（2-3 周）

**目标**：深入理解 table 和元表，掌握 Lua OOP 实现。

- table 库（sort/concat/insert/remove/move）
- 元表：`__index`、`__newindex`、`__call`、`__tostring`
- 运算符重载
- 基于原型的类实现
- 继承与多重继承
- 闭包封装私有状态

**里程碑**：实现一个完整的 Vector/Complex 数学库（包含运算符重载）。

**推荐资源**：**《Programming in Lua》第 11-13, 16 章**。

## 阶段三：模块与协程（2-3 周）

**目标**：掌握模块系统和协程，理解 Lua 的非抢占式调度模型。

- require 机制与 package 系统
- 模块组织与发布（LuaRocks）
- 协程：create/resume/yield
- 生产者-消费者、迭代器模式
- 简易协程调度器
- 错误处理：pcall/xpcall

**里程碑**：实现一个基于协程的简易异步 TCP 服务器（luasocket + 协程）。

**推荐资源**：**《Programming in Lua》第 9, 21, 23 章**。

## 阶段四：C API 与嵌入（3-4 周）

**目标**：掌握 Lua 与 C/C++ 的双向交互。

- Lua C API 基础（`lua_State`、栈操作）
- 注册 C 函数给 Lua
- userdata 和元表（C 侧）
- 从 C 调用 Lua 函数
- 错误处理和栈保护
- LuaJIT FFI（直接调用 C 库）

**里程碑**：为 C 程序添加 Lua 脚本扩展能力（暴露 C 函数、创建 userdata）。

**推荐资源**：**《Programming in Lua》第 24-25 章**，**LuaJIT FFI 教程**。

## 阶段五：应用领域深入（3-6 个月）

**目标**：在特定领域深耕 Lua 实战能力。

- **游戏方向**：LÖVE 引擎、行为树 AI、游戏 UI 脚本
- **Web 方向**：OpenResty（Nginx + LuaJIT）、Redis Lua 脚本
- **工具方向**：Neovim 插件开发

**里程碑**：发布一个开源 Lua 项目（游戏、插件或库）。

**推荐资源**：**LÖVE 官方 Wiki**、**OpenResty 最佳实践**、**Neovim Lua Guide**。

## 持续学习的资源

- lua.org — 官方文档和参考手册
- luajit.org — LuaJIT 文档和优化指南
- lua-users.org/wiki — 社区知识库
- OpenResty 官方博客 — 高性能 Lua 实战案例
