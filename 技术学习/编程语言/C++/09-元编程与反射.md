# C++ 元编程与反射

## 详细描述

C++ 的元编程能力是世界顶级的——模板系统本身就是一个图灵完备的编译期函数式编程语言。但代价是复杂性，需要耐心掌握。

**模板元编程**（Template Metaprogramming, TMP）利用模板的递归实例化在编译期执行计算。经典的斐波那契、编译期质数判断都是 TMP 的练习。C++11 的变参模板和 C++17 的折叠表达式大幅降低了 TMP 的代码复杂度。

**constexpr/consteval** 将元编程从模板的"函数式"风格带到了更自然的"命令式"风格。`constexpr` 函数在编译期上下文中求值时就像普通函数一样——可以用循环、条件、变量等常规语法。`consteval`（C++20）更进一步：保证函数**必须**在编译期求值。这是元编程范式的转变。

**type_traits**（`<type_traits>`）提供了编译期类型检查和变换的标准组件。`std::is_integral`、`std::is_same`、`std::is_base_of` 等用于类型判断；`std::remove_reference`、`std::decay`、`std::underlying_type` 等用于类型变换；`std::enable_if` 基于条件启用/禁用模板。

**if constexpr**（C++17）在编译期选择分支，未选择的分支不被实例化。这让代码在同一模板中根据类型条件采用不同实现，而不需要多个重载或 SFINAE。

**无运行时反射**是 C++ 的一个重要限制。与其他主流语言不同，C++ 没有内建的运行时反射（不能遍历类的成员、获取类型名称等）。这一直是 C++ 标准委员会讨论的热点。当前解决方案：
- 宏/代码生成（手动写反射宏，或使用 Qt MOC）
- 第三方库（Boost.Hana、rttr）
- C++26 可能的 `std::meta`（编译期反射，可生成代码）

**`static_assert` 与编译期诊断**：在编译期检查条件并提供友好的错误信息。与 type_traits 配合是构建良好模板接口的关键。

## 代码实例

```cpp
#include <iostream>
#include <type_traits>
#include <array>

// ========== 1. 模板元编程 ==========

// 编译期阶乘
template <int N>
struct Factorial {
    static constexpr int value = N * Factorial<N - 1>::value;
};
template <>
struct Factorial<0> {
    static constexpr int value = 1;
};

constexpr int factorial7 = Factorial<7>::value;  // 5040

// 编译期判断是否为质数
template <int N, int D>
struct IsPrimeHelper {
    static constexpr bool value = (N % D != 0) && IsPrimeHelper<N, D - 1>::value;
};
template <int N>
struct IsPrimeHelper<N, 1> {
    static constexpr bool value = true;
};
template <int N>
struct IsPrime {
    static constexpr bool value = IsPrimeHelper<N, N - 1>::value;
};

// ========== 2. constexpr 命令式元编程 ==========

// 现代 constexpr 元编程：比TMP直观得多
constexpr int factorial_modern(int n) {
    int result = 1;
    for (int i = 2; i <= n; ++i) result *= i;
    return result;
}

constexpr int fact10 = factorial_modern(10);  // 3628800

// consteval：强制编译期求值
consteval int compile_only(int n) {
    return n * n;
}

constexpr int x = compile_only(10);  // OK
// int y = compile_only(x);         // OK（x 是 constexpr）
// int z = compile_only(rand());    // 错误！运行时值不能传参给 consteval

// 编译期字符串处理
constexpr size_t string_length(const char* s) {
    size_t len = 0;
    while (s[len] != '\0') ++len;
    return len;
}

static_assert(string_length("hello") == 5);

// ========== 3. type_traits ==========
template <typename T>
void describe_type() {
    if constexpr (std::is_integral_v<T>) {
        if constexpr (std::is_signed_v<T>) {
            std::cout << "signed integer, size: " << sizeof(T) << "\n";
        } else {
            std::cout << "unsigned integer, size: " << sizeof(T) << "\n";
        }
    } else if constexpr (std::is_floating_point_v<T>) {
        std::cout << "floating point, size: " << sizeof(T) << "\n";
    } else if constexpr (std::is_class_v<T>) {
        std::cout << "class/struct\n";
    } else {
        std::cout << "other type\n";
    }
}

// 类型变换
template <typename T>
using CleanType = std::remove_cvref_t<std::remove_pointer_t<T>>;

// ========== 4. 宏模拟反射 ==========

// 手动反射宏
#define REFLECT_MEMBER(TYPE, NAME) \
    { #NAME, offsetof(TYPE, NAME), sizeof(TYPE::NAME) }

struct Person {
    std::string name;
    int age;
    double height;
};

// 为 Person 注册反射信息
struct FieldInfo {
    const char* name;
    size_t offset;
    size_t size;
};

constexpr FieldInfo person_fields[] = {
    REFLECT_MEMBER(Person, name),
    REFLECT_MEMBER(Person, age),
    REFLECT_MEMBER(Person, height),
};

void reflect_example() {
    Person p{"Alice", 30, 1.65};
    
    for (const auto& field : person_fields) {
        std::cout << "Field: " << field.name 
                  << ", offset: " << field.offset
                  << ", size: " << field.size << "\n";
    }
}

// ========== 5. 编译期容器（C++20） ==========
constexpr auto make_fib_array(int n) {
    std::array<int, 20> result{};
    result[0] = 0;
    result[1] = 1;
    for (int i = 2; i < n; ++i) {
        result[i] = result[i - 1] + result[i - 2];
    }
    return result;
}

constexpr auto fib_values = make_fib_array(15);
static_assert(fib_values[10] == 55);

// ========== 6. C++26 概念：std::meta ==========
// （未来特性，展示概念）
// #include <experimental/meta>
// template <typename T>
// void print_members(T const& obj) {
//     for... (auto member : std::meta::nonstatic_data_members_of(^T)) {
//         std::cout << std::meta::name_of(member) << " = " << obj.[:member:] << "\n";
//     }
// }

int main() {
    describe_type<int>();
    describe_type<double>();
    describe_type<std::string>();
    
    reflect_example();
    
    for (int i = 0; i < 15; ++i) {
        std::cout << fib_values[i] << " ";
    }
    std::cout << "\n";
    
    return 0;
}
```

## 推荐书籍

1. **《C++ Templates: The Complete Guide》（第2版）** — David Vandevoorde / Nicolai Josuttis
   第 19-28 章深入讲解模板元编程、type_traits、SFINAE 和变参模板，是 TMP 的百科全书。

2. **《Template Metaprogramming with C++》** — Marius Bancila
   从 SFINAE 到 concept 的全面 TMP 实战，覆盖 enable_if、if constexpr、折叠表达式等现代技术。

3. **《C++20 - The Complete Guide》** — Nicolai M. Josuttis
   consteval、编译期 string/vector、改进的 constexpr 等 C++20 元编程相关特性有详细讲解。
