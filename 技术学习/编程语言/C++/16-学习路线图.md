# C++ 学习路线图

## 阶段一：基础入门（1-2 个月）

**目标**：能用 C++ 写简单程序，理解基本语法和编译流程。

- 基本语法：变量、类型、运算符、控制流
- 指针与引用（核心难点，需要反复练习）
- 函数：声明、定义、重载、默认参数、const 成员函数
- 类的基础：构造/析构、访问控制、成员函数
- 标准输入输出、文件操作
- 编译链接流程（gcc/clang/msvc）

**里程碑**：实现一个命令行通讯录程序（增删改查，文件存储）。

**推荐资源**：**《C++ Primer》第 1-7 章，15 章**。

## 阶段二：面向对象与 STL（2-3 个月）

**目标**：掌握 C++ OOP 和标准库，能编写中等复杂度的程序。

- 继承、虚函数、多态、抽象类
- 运算符重载
- STL 容器：`vector`、`map`、`set`、`list`
- STL 算法：`sort`、`find`、`transform`、`accumulate`
- 迭代器与 range-for
- 异常处理：`try/catch`、RAII 理念

**里程碑**：实现一个简易的学生成绩管理系统（用 STL 容器和算法）。

**推荐资源**：**《C++ Primer》第 8-18 章**，**《Effective C++》第 1-20 条**。

## 阶段三：现代 C++（2-3 个月）

**目标**：掌握 C++11/14/17 核心特性，写出地道的现代 C++ 代码。

- 智能指针（`unique_ptr`、`shared_ptr`、`weak_ptr`）
- 移动语义与完美转发
- Lambda 表达式与 `std::function`
- `auto`/`decltype`/`decltype(auto)`
- `constexpr`/`if constexpr`
- `std::optional`/`std::variant`/`std::any`
- `std::string_view`
- 并发基础：`std::thread`、`std::mutex`

**里程碑**：实现一个多线程文件下载器。

**推荐资源**：**《Effective Modern C++》全部**，**《A Tour of C++》**。

## 阶段四：模板与元编程（2-3 个月）

**目标**：理解模板机制和编译期编程，能阅读和编写模板库。

- 函数模板与类模板、模板特化
- 变参模板与折叠表达式（C++17）
- `type_traits` 与 `enable_if` vs `if constexpr`
- `concept`（C++20）
- SFINAE 基础
- 简单的模板元编程

**里程碑**：实现一个类型安全的泛型序列化库。

**推荐资源**：**《C++ Templates》**，**《C++20 - The Complete Guide》concept 章节**。

## 阶段五：系统与底层（3-6 个月）

**目标**：深入理解 C++ 与 OS 的交互，具备系统级开发能力。

- 内存管理：自定义 allocator、内存映射、缓存对齐
- 高级并发：协程（C++20）、无锁数据结构、内存序
- 网络编程：Boost.Asio 或 Seastar 框架
- 调试与性能分析：GDB、perf、Valgrind/ASan

**里程碑**：实现一个基于 epoll/IOCP 的异步网络服务器。

**推荐资源**：**《C++ Concurrency in Action》**，**《Systems Performance》**。
