# C++ 常用框架与生态

## 详细描述

C++ 的框架生态横跨多个领域——从系统级库到高性能网络框架，从图形渲染到科学计算。

**Boost** 是 C++ 最重要的第三方库集合，素有"准标准库"之称。许多 Boost 组件最终进入了 C++ 标准（如 `shared_ptr`、`filesystem`、`asio`、`variant`）。核心组件：`Boost.Asio`（异步网络 I/O）、`Boost.Beast`（HTTP/WebSocket）、`Boost.Spirit`（解析器生成）、`Boost.Graph`（图算法）、`Boost.Python`（C++/Python 绑定）。

**Qt** 是跨平台 GUI 框架的王者，也是 C++ 最成功的应用程序框架之一。不仅提供了完整的 UI 组件，还包括网络、数据库、XML、WebEngine 等模块。QML（Qt Modeling Language）支持声明式 UI 开发。Qt Creator 是一流的集成开发环境。

**Seastar** 是 ScyllaDB 开发的异步无共享（shared-nothing）C++ 框架。它基于未来/承诺（future/promise）模型和用户态 TCP/IP 栈，以单个核心一个线程的方式最大化现代多核 CPU 的性能。

**OpenCV** 是计算机视觉领域的标准库，提供图像处理、视频分析、目标检测、机器学习的丰富算法。

**Unreal Engine** 使用 C++ 作为核心开发语言，配合蓝图（Blueprint）可视化脚本。UE 的 C++ 扩展了大量编译宏和反射系统（UCLASS/USTRUCT/UFUNCTION）。

## 代码实例

```cpp
// ========== Boost.Asio 异步服务器 ==========
#include <boost/asio.hpp>
#include <iostream>

using boost::asio::ip::tcp;

class SimpleSession : public std::enable_shared_from_this<SimpleSession> {
    tcp::socket socket_;
    std::array<char, 1024> buffer_;

public:
    explicit SimpleSession(tcp::socket socket) : socket_(std::move(socket)) {}

    void start() { do_read(); }

private:
    void do_read() {
        auto self(shared_from_this());
        socket_.async_read_some(boost::asio::buffer(buffer_),
            [this, self](boost::system::error_code ec, size_t length) {
                if (!ec) {
                    do_write(length);
                }
            });
    }

    void do_write(size_t length) {
        auto self(shared_from_this());
        boost::asio::async_write(socket_, boost::asio::buffer(buffer_, length),
            [this, self](boost::system::error_code ec, size_t) {
                if (!ec) do_read();
            });
    }
};
```

## 推荐书籍

1. **《The Boost C++ Libraries》** — Boris Schäling
   逐一讲解 Boost 中最重要的库，每个库都有清晰的示例和使用动机。
2. **《C++ GUI Programming with Qt 6》** — Lee Zhi Eng
   Qt 6 的权威入门书，从 Widget 到 QML 到网络编程。
3. **《Mastering OpenCV 4》** — Roy Shilkrot / David Millán Escrivá
   以项目驱动的方式讲解计算机视觉的实战应用。
