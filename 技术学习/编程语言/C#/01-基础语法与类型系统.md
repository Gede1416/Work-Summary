# C# 基础语法与类型系统

## 详细描述

C# 的类型系统融合了静态类型的安全性和类型推导的简洁性，分为**值类型**和**引用类型**两大体系，这是理解 C# 内存模型和性能特征的基础。

**值类型**（struct、enum、基本数值类型）直接存储数据本身，分配在栈上（或作为对象的一部分在堆上），赋值时发生完整拷贝。值类型不可为 null（除非声明为可空类型 `T?`），适合表示小型、不可变的数据结构。

**引用类型**（class、interface、delegate、record、string、array）存储的是指向堆内存的引用，赋值时只拷贝引用而非数据。引用类型默认为 null，C# 8.0 引入的**可空引用类型**（NRT）可以在编译期追踪潜在的 null 引用，极大减少了 NullReferenceException。

**可空值类型**（`T?`）是 `Nullable<T>` 的语法糖，允许值类型表示"无值"状态。配合 `??`（空合并运算符）和 `?.`（空条件运算符）可以写出简洁的空安全代码。

**元组**（C# 7.0）提供了轻量级的数据聚合方式，不需要定义类或结构体。配合解构语法，可以让方法"返回多个值"。`ValueTuple` 是值类型实现，性能优于 `Tuple` 引用类型。

**记录**（record，C# 9.0）是一种引用类型（`record class`）或值类型（`record struct`，C# 10），专为不可变数据建模设计。记录自动生成 `Equals`、`GetHashCode`、`ToString` 等方法，支持 `with` 表达式创建修改后的副本。

**模式匹配**（C# 7.0-11）是 C# 近年最重要的语法演进：类型模式、属性模式、位置模式、关系模式、逻辑模式（`and/or/not`），以及列表模式（C# 11）。让条件逻辑表达力大幅提升。

**顶层语句**（C# 9.0）消除了 `Main` 方法的样板代码，让小程序和脚本更简洁。**init 访问器**（C# 9.0）允许属性在对象初始化时设置，之后变为只读。

**Span\<T\> 和 Memory\<T\>** 提供了对连续内存的安全抽象，可以在不分配堆内存的情况下操作数组切片、栈内存和托管/非托管内存。

## 代码实例

```csharp
// ========== 1. 值类型与引用类型 ==========
// 值类型：struct
struct Point
{
    public int X { get; set; }
    public int Y { get; set; }
    
    public Point(int x, int y) => (X, Y) = (x, y);
}

// 引用类型：class
class Person
{
    public string Name { get; set; }
    public int Age { get; set; }
}

// 记录：不可变引用类型（C# 9.0+）
record PersonRecord(string Name, int Age);

// 记录结构体：不可变值类型（C# 10+）
readonly record struct PointRecord(int X, int Y);

// ========== 2. 可空类型与空安全 ==========
void NullableTypes()
{
    // 可空值类型
    int? maybeNumber = null;
    maybeNumber ??= 42;  // 如果为 null 则赋值
    
    // 空合并运算符
    int definiteNumber = maybeNumber ?? 0;
    
    // 空条件运算符
    Person? person = null;
    string? name = person?.Name;  // null，不会抛异常
    
    // 可空引用类型（NRT，C# 8.0+，需在 .csproj 中启用）
    // <Nullable>enable</Nullable>
    string nonNullable = "hello";  // 不能为 null
    string? nullable = null;       // 可以为 null
}

// ========== 3. 元组与解构 ==========
void Tuples()
{
    // 具名元组
    (string Name, int Age) person = ("Alice", 30);
    Console.WriteLine($"{person.Name} is {person.Age}");
    
    // 方法返回元组
    (int min, int max) FindMinMax(int[] numbers)
    {
        return (numbers.Min(), numbers.Max());
    }
    
    var (min, max) = FindMinMax(new[] { 1, 5, 3, 9, 2 });
    Console.WriteLine($"Min: {min}, Max: {max}");
}

// ========== 4. 模式匹配 ==========
void PatternMatching()
{
    object obj = 42;
    
    // 类型模式
    if (obj is int number)
        Console.WriteLine($"整数: {number}");
    
    // 属性模式
    var person = new Person { Name = "Bob", Age = 25 };
    string description = person switch
    {
        { Age: >= 18 } => "成年人",
        { Age: < 18 }  => "未成年人",
        _               => "未知"
    };
    
    // 关系模式 + 逻辑模式（C# 9.0+）
    static string Classify(int x) => x switch
    {
        < 0              => "负数",
        0                => "零",
        > 0 and < 10     => "个位数",
        >= 10 and < 100  => "两位数",
        _                => "大数"
    };
    
    // 列表模式（C# 11）
    static string DescribeArray(int[] arr) => arr switch
    {
        []        => "空数组",
        [int f]   => $"单元素: {f}",
        [int f, .., int l] => $"首: {f}, 尾: {l}",
        _         => "其他"
    };
}

// ========== 5. Span<T> 零分配切片 ==========
void SpanExamples()
{
    // Span<T> - 栈上的内存视图
    Span<int> numbers = stackalloc int[] { 1, 2, 3, 4, 5 };
    
    // 切片操作（零分配）
    Span<int> slice = numbers[1..4];  // [2, 3, 4]
    slice[0] = 99;  // numbers 变为 [1, 99, 3, 4, 5]
    
    // 在数组上使用 Span
    int[] array = new int[100];
    Span<int> arraySpan = array.AsSpan(10, 30);  // 数组切片
    
    // ReadOnlySpan<T> 保证只读
    ReadOnlySpan<char> text = "Hello, World!".AsSpan();
    ReadOnlySpan<char> world = text[7..12];  // "World"
}

// ========== 6. 顶层语句（C# 9.0+） ==========
// 在 Program.cs 中可直接写：
// Console.WriteLine("Hello, World!");
// 无需显式 Main 方法和 using 声明（隐式 using）

// ========== 7. init 访问器（C# 9.0+） ==========
class Configuration
{
    public string Name { get; init; }   // 对象初始化时设置，之后只读
    public int Timeout { get; init; } = 30000;
}

var config = new Configuration { Name = "MyApp", Timeout = 5000 };
// config.Name = "Other";  // 编译错误！init-only 属性只能在初始化时设值
```

## 推荐书籍

1. **《C# 12 in a Nutshell》** — Joseph Albahari
   C# 权威参考书，覆盖 C# 12 及 .NET 8，从基础语法到高级特性一应俱全，是每个 C# 开发者案头必备的工具书。

2. **《C# in Depth》（第4版）** — Jon Skeet
   Stack Overflow 传奇人物 Jon Skeet 的经典之作，深入 C# 语言演进的历史和设计哲学，让你不仅会用，更懂为什么。

3. **《Pro C# 10 with .NET 6》** — Andrew Troelsen / Phil Japikse
   全面覆盖 C# 基础和 .NET 运行时，从类型系统到高级主题，适合系统性学习，内容详实且包含大量示例。

4. **《CLR via C#》（第4版）** — Jeffrey Richter
   深入 .NET CLR 内部机制，理解值类型/引用类型、GC、JIT 等底层原理，是进阶 C# 开发者的必读书。
