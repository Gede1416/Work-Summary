# C# 常用框架与生态

## 详细描述

C# 和 .NET 的框架生态以微软官方支持为核心，辅以庞大的开源社区，覆盖了从 Web 到桌面、从移动端到游戏开发的全栈场景。

**ASP.NET Core** 是 .NET 的 Web 应用框架，跨平台、高性能。核心组件：MVC（模型-视图-控制器）、Minimal API（最简 HTTP API）、Razor Pages（服务端渲染页面）、SignalR（实时 WebSocket 通信）、gRPC 服务、认证与授权中间件。Kestrel 是内建的跨平台 Web 服务器，性能在 TechEmpower 基准测试中名列前茅。

**Entity Framework Core** 是 .NET 的 ORM（对象关系映射），支持 Code First（从代码生成数据库）、Database First（从数据库生成代码）、Migrations 数据库迁移。支持多种数据库：SQL Server、PostgreSQL、MySQL、SQLite、Cosmos DB。

**WPF**（Windows Presentation Foundation，Windows 桌面）和 **MAUI**（.NET Multi-platform App UI，跨平台桌面+移动）是桌面应用开发的主要框架。XAML 声明式 UI，MVVM 架构模式。

**Unity** 游戏引擎使用 C# 作为脚本语言，是全球最流行的游戏引擎之一。Unity 的 C# API 覆盖 2D/3D 渲染、物理、动画、UI、网络等。

**Blazor** 使 C# 能在浏览器中运行——通过 WebAssembly。Blazor Server 基于 SignalR 的实时模式；Blazor WebAssembly 直接在浏览器中运行 .NET 代码。

## 代码实例

```csharp
// ========== ASP.NET Core Minimal API ==========
var builder = WebApplication.CreateBuilder(args);
var app = builder.Build();

app.MapGet("/hello/{name}", (string name) => $"Hello, {name}!");
app.MapPost("/items", (Item item, AppDbContext db) => {
    db.Items.Add(item);
    db.SaveChanges();
    return Results.Created($"/items/{item.Id}", item);
});

app.Run();

// ========== EF Core 实体 ==========
[Table("Users")]
public class User
{
    public int Id { get; set; }
    
    [Required]
    [MaxLength(100)]
    public string Name { get; set; } = "";
    
    public List<Order> Orders { get; set; } = new();
}

// ========== Blazor 组件 ==========
@* @page "/counter"
<h1>Counter</h1>
<p>Current count: @currentCount</p>
<button @onclick="IncrementCount">Click me</button>

@code {
    private int currentCount = 0;
    private void IncrementCount() => currentCount++;
} *@
```

## 推荐书籍

1. **《ASP.NET Core in Action》（第3版）** — Andrew Lock
   从入门到高级的系统讲解，覆盖 MVC、Razor Pages、Web API、SignalR、认证，是 ASP.NET 的最佳学习路径。
2. **《Entity Framework Core in Action》（第2版）** — Jon P Smith
   深入 EF Core 的内部机制和最佳实践，包括查询优化、并发处理、领域驱动设计。
3. **《C# 12 in a Nutshell》** — Joseph Albahari
   LINQ、EF Core、ASP.NET Core、HTTP 客户端等相关章节提供了可靠的框架参考。
